 </span>五星， 电影“真实”的解读，感觉比《剧本》更重要，不仅展示了戈达尔的创作理念，更是他用影像和思想进行的最后一次自我剖析与艺术反思。

](https://movie.douban.com/subject/1465471/)## [幽灵马车 (1921)
](https://movie.douban.com/subject/1465471/)[8.4
1921 / 瑞典 / 剧情 恐怖 奇幻 / 维克多·斯约斯特洛姆 / 维克多·斯约斯特洛姆 希尔达·博里斯特伦
](https://movie.douban.com/subject/1465471/) <span style="text-align: center;">
     遮罩和灵魂下才是真实

 </span>电影剧情太感人了，除了一个美好结尾不太让人满意，其他的可以说相当完美。电影改编自瑞典的一部小说，其中主要的内核是救赎与自省，结构还是以奥德赛之旅的公路片。一部二几年的作品的叙事形式放到现在依然是相当炸裂的存在，以开场做结尾，中途夹杂了心灵之旅。
视觉上，数字修复的效果还不错，尤其是大量双爆场景的呈现出的游魂状态太棒了。剪辑效果和构图也都非常棒，只有少有的几场跃轴有些不适，冷暖色的呈现也非常符合情节，最吸引我的是遮罩的使用，尤其是很多时候聚焦于角色的表情类似于变焦的强调效果，感觉很大程度完成了对角色的塑造，为结尾做了一个不错的铺垫。
听觉上，Steven Horne的现场配乐我实在太喜欢了，其实看这部电影很大一部分原因也是因为是他的配乐。他最常用的一种技法应该是两件乐器的同时演奏，感觉这种叠加混音呈现出的效果太棒了，尤其是他很明显的研究了剧情和情绪，所以配乐很多时候都很能帮助表达。整部电影中他使用的最多的肯定还是钢琴，尤其是很多时候的抒情呈现就是完整的演奏，琴音结合着修女那部分的剧情实在是太催泪了。影片中还有很多时候是结合钢琴与其他乐器呈现出别样的效果，例如类似迷你木琴和钢琴的结合，这部分主要是呈现修女病重时的状态，木琴清脆单纯的高音如同修女的内心呈现，而钢琴低沉的音调则是她身体的状态。还有手风琴结合钢琴的共同演奏，这里我感觉有三种状态的呈现，高音的手风琴基本呈现暴怒的情绪状态而钢琴作为底音，不拉只按则是偏向欢快，最后是模拟喘气的空拉的音效呈现。最后则是长笛结合钢琴，这里基本是用于呈现幽灵状态，基本每次都是灵魂对谈的时候使用，其实是完全没有恐怖的氛围呈现，更多的是偏向悠扬的旋律来体现那种奇幻感。影片中不光用乐器呈现了完美的配乐，也完成了很多音效的表达，包括利用拨扫钢琴的琴弦造成或尖锐或强力的音效来呼应剧情，如马车车轮停止时尖锐的声音，撕扯衣服时的声音等，而另一个类似于铃声的打击乐也是完整的模仿了钟表整点报时的声音，甚至还能感觉到那种远景减弱的感觉！
致敬该影片的电影实在太多了，最明显的肯定是博格曼的电影作品，《第七封印》中的死神，甚至《野草莓》中的钟表与虚幻的呈现（更不用说《野草莓》的男主就是这部电影的导演和男主了）和《闪灵》中的斧头劈门，还有一些比较有趣的感觉，类似于两个小孩背对门那一幕让我想起了《盗梦空间》，修女的状态想起了《呼喊与细语》，双灵魂的旅行想起了《此房是我造》，故事想起了《生活多美好》。
其实看到这部电影放映的时候票已经在官网上无法购买了，显示的是少量存票，可以提前到现场购票或者发邮件，影院的地点还是有些偏，之前去看《贼博士》完了后也属于抹黑回家的状态，象堡也是号称伦敦小哥谭，而且本来我就怕麻烦所以打算放弃了，但当我就注意到了现场配乐的钢琴家是Steven Horne时我毫不犹豫的发了一封邮件冲了（虽然人家没回，我还是提早去现场买了票），我看过现场配乐的电影大概有15部左右，但是给我印象最深的就是Steven Horne，能感觉到他很明显的做了很多功课，对于如何呈现某个时刻，某种情绪都有很独特的理解及呈现、看过《诱惑》《群众》《少奶奶的扇子》都是他配乐的，而且我都超级喜欢！还有另一位默片配乐师John Sweeney，看过《浮草物语》和上下两部的《玩家马布斯博士》印象也是极为深刻，今天特别巧合的是在《幽灵马车》放映前还放映了一部梅里爱的《仙女国》正好是John Sweeney现场配乐（介绍里也没写），效果也非常好！我属于一天看了两部我最喜欢的两位配乐师配乐的电影。
稍微不喜欢的可能是结尾的呈现和犹豫一人演奏切换乐器时中间造成音效戛然而止的感觉，不过完全可以理解。

](https://movie.douban.com/subject/1304857/)## [我与长指甲 (1987)
](https://movie.douban.com/subject/1304857/)[7.7
1987 / 英国 / 剧情 喜剧 / 布鲁斯·罗宾逊 / 理查德·E·格兰特 保罗·麦甘恩
](https://movie.douban.com/subject/1304857/) <span style="text-align: center;">
     There must be some kind of way outta here

 </span> 影片中那种醉酒的感觉我不曾拥有，也不想去尝试，似乎我知道我很容易就成瘾般从而拒绝尝试，实际上我连不清醒的时候都非常的少，所以很难想象但恰恰我却很喜欢这部电影，仿佛让我体会到了那种不曾拥有的迷幻感，这种感觉在看《追讨者》的时候也出现过，但在这部电影中格外强烈，可能是因为那种潮湿的空气很熟悉，或许早上摄政公园冷清让人精神百倍，也可能是因为那中挣脱出的勇气让人敬佩，亦或者是长指甲消失在倾盆大雨中的那坚定的背影。
电影讲述了六十年代末英国的两个失业的演员的生活状态，内容是英式幽默的喜剧，比较可惜的是其中的太多双关语并不能理解到跟全场同时发笑，但其中大量的情节的无厘头程度也是足够好笑了，最好笑的是影院在放映前组织了酒会。
影片的质感让我想到了另一部影片《我自己的爱达荷》，同样的公路电影，同样的两个角色，同样的复杂且微妙的关系塑造，同样的边缘化人物，同样的颓废质感，同样的没有方向，同样的一方离去作为结束，但不知道为什么他们两个人的离去带来的的感受却似乎是相反的，《长指甲》中能明显感觉到那种反思，那种对于反叛精神和嬉皮士文化的反思，但同时也如此浪漫的设计了一个结尾，就仿佛他选择清醒了，但却对那个长指甲有着一股深深的迷恋。导演罗宾逊以自己的亲身经历为原型设计了“我”，而长指甲则是基于他的一位朋友Vivian MacKerrell。影片的风格让我感觉很大程度的跟林赛安德森的作品有关系。林赛的作品中大量对社会尖锐批判和阶级讽刺，而影片更像是承袭这个反叛的结果后所产生的反思，去乡下是逃离，回到城市依然是逃离，影片就像是一个做好决定的人由迷茫的站在十字路口，一个何去何从的反思。
视觉上，黏腻是给我最深的感觉，不论是情感还是环境，镜头是中规中矩的，低角度仰拍的几场戏的构图很喜欢，英国阴冷的天气整体的配色就偏冷色调，接近夜幕行走的那场戏和天光初亮的那两场戏超级爱，雾蒙蒙的天气也让我想到了那幅画《海雾中的漫游者》，实在是太美了。
听觉上，摇滚乐强化了时代的背景，柔和俏皮的音效很符合电影的风格，当然还有大量环境音的参与。

](https://movie.douban.com/subject/1298736/)## [甜妹妹 (1989)
](https://movie.douban.com/subject/1298736/)[7.3
1989 / 澳大利亚 / 剧情 喜剧 / 简·坎皮恩 / Geneviève Lemon Karen Colston
](https://movie.douban.com/subject/1298736/) <span style="text-align: center;">
     树上那破碎的心

 </span>电影用极其复杂和隐晦的方式呈现出了一个家庭的状态，影片中没有明确展现乱伦，但具有强烈的乱伦暗示，家庭成员间的关系复杂且非常扭曲，尤其是姐姐与父亲之间那种略有病态的呈现，而女主作为另一面呈现出了该家庭成长中的另一种病态，甜妹是直接受害者，而女主则像是旁观的间接受害者，这也解释了两位女性角色的中和可能是一个相对稳定的性格呈现，而影片本质是一位女性在异化社会中何去何从的呈现，那种情感的张力和联结呈现出的效果是来自一种对于漠视的讽刺，感觉导演对于个体在异化的家庭或社会上的表现呈现出极细腻的理解和观察，《犬之力》感觉也是类似的表达，也是因为看了《犬》那同样的精致视觉表达让我开始认真写影评的。（其实影片总让我想到是一个更现实版的《博很恐惧》）
意象与隐喻上，主要是两点，树和马，影片中树其实是贯穿了整部影片的，在我看来树象征的是不好的事儿，是性侵与死亡。影片开场的独白就强调了树，“树是甜妹的，上面有宫殿是为了她而造的，她是公主“（这里的配乐也是极其低沉和压抑的），她从来不让女主上去，并且女主恐惧树，这里的树在我看来是父亲性侵甜妹的地点（后续会用红色佐证，其实可以看到父亲在家里好像一直在甜妹的房间），这也证明甜妹在很大程度上保护了女主，而女主作为间接的受害者的表达更多的是作为间接受害者的恐惧表达，这也解释了男主想在家里种树时女主对于树的处理，而这里的树则有死亡之意，因为这个树被女主拔掉后，被填平后上面摆放着小男孩的“小车”似乎对应了结尾处甜妹入葬时的处理，同样的推土机填平。马，感觉更多的是被抽离的欲望，甜妹的吞噬本身是一种汲取，而女主在甜妹离世之后才选择有所释放。马的名字也非常有意思，印象最深的是最后一匹马，破碎的心，也就是甜妹。
抛开内核表达影片最喜欢的应该是角色的塑造和视觉上的呈现。
角色上，女主角和她的妹妹之间存在着极复杂的关系，女主内向且焦虑，而甜妹则表现出明显心理问题，父亲的破碎的父权和母亲的无所作为则是推动这一切的根源，感觉黑色的意象很大程度的展现了家庭曾经的巨大变革。甜妹的演绎非常棒，把那种癫狂戏谑演绎的淋漓尽致，那种失控的状态更是极尽真实。
视觉上，首先想先提及下红色，红色在影片中是性的代表，从开场的停车场的条纹红色让女主与男主在一起，到后续男主同事勾引他是旁边的红色灯，再到之后餐厅中鲍勃与女服务员间的红色椅子，当然这一切到最后就可以看到父亲是使用红色的桶给甜妹运食物，这也能作证乱伦关系，最后的结尾女主的床垫是红色的，也预示了男女主间的性关系。总感觉黑色涂料也有所指代，因为记得她是先从手变黑的，到之后才涂满全身。

     树，画，电影和我

 </span>人画树，导演导电影，我观影。影片整体呈现的效果太好了，不论试听还是节奏亦或者呈现的内容。“画”的过程感觉也是对照了电影拍摄的过程，其实也对照了工人施工。构图（影片内外前期的准备），视角（画家双脚站立的位置也摄影机架设的位置），元电影（画家与摄影机），时间的流逝（内的果实上的刻度，外的四季变化），和人生（作为客体躺下与果实坠落腐烂）。那躺在床上的男主和关掉的灯也对照了之前男主放到地下室的画作和关上的挡板。太喜欢这个结局的递进关系了，摄影机拍摄，带着标记的果实腐烂，又一年春。之前还在想那个低画质的画面呈现来自于哪里，到结尾一切都明了了。
影片中其实“呈现”了很多次树，去年画家的树在对话中出现，今年的树有两张画作，还有妻子工作刻板的树，甚至装修，电影的拍摄。整体的时间流动概念让我想起了古希腊哲学家赫拉克利特的名言“人不能两次踏入同一条河流”，也想到了之前看过一个公开课中的一个老师说的“同一个人不能两次踏入同一条河流”的延展，画家也不是前一刻的画家了，而两个老友讨论的米开朗基罗也是对照了两个人如何看待“画家”，感觉非常像“画家”的自辨，其中也谈到米开朗基罗把自己画进画里感觉也对应了那棵树及这部电影，而与此有关的另一个思考就是影片的真实性，记录传记片的现实与演绎也被模糊的非常好，很难想到情绪，天气与一切的来访的真实性。
艺术作品，维纳斯的雕像，残缺的完美与画作中完美的残缺对应。片中还出现了胜利女神雕像的画像和家庭相关的画作在结尾。
视觉上，只能说是精致…尤其是构图，越过树和画框…垂直和平行的两条线构建出的空间实在是太美了。光影也一如既往的好，冷暖色的变化展现时间，利用抽烟呈现情绪，还有那现实的高楼大厦的空镜，提醒着我们现实的世界，这么一想那关掉的电视又何尝不是闭上眼睛呢。镜子的几次使用都很喜欢。
听觉上，基本都是现场收音，音效收进来的效果太棒了，配乐也很不错，台词还是挺少的，中国访客到来的时候问的那个问题还蛮有趣的，是否可以在果实掉落前画完素描呢？“we will see”仿佛是在对观众的回答，结尾我们也确实看到了。
看到影片首先想到的是里维特的《不羁的美女》，都是带有匠人属性，但是风格上还是相差很大。折页上也提到了彼得·格林纳威的《一个Z和两个O》中向我们展示了一只死去的斑马的加速腐烂，看电影的时候我也感觉到这种艺术性收尾和部分风格有些类似格林纳威的作品，但一时却想不起名字。

 </span>超喜欢的一部作品，戏中戏设定，晚于戈达尔那部超级讽刺的《蔑视》，早于特吕弗那部奔向好莱坞的《日以作夜》，整体以三面表达了那纠缠的爱情。叙事整体看是顺时叙事，但仔细想想影片的开场好像就会有女主在火车上的镜头，中途还有插入类似的回忆，这不光让影片的结构更完整，前后有呼应，也同时也加强了男女主爱情的宿命感。

](https://movie.douban.com/subject/1476347/)[7.4
1995 / 美国 / 剧情 恐怖 / 阿贝尔·费拉拉 / 莉莉·泰勒 克里斯托弗·沃肯
](https://movie.douban.com/subject/1476347/) <span style="text-align: center;">
     超脱命定的自由意志

 </span>看完影片让我最先想到的是两部以冷战为背景同时都带有类似哲学理念的电影作品，布列松的《很可能是魔鬼》以美苏军备竞赛为背景展现了一个宿命论般的故事，作为学生的男主被虚无主义包裹后的自杀（也是谋杀），和柯南伯格的《裸体午餐》以间谍身份隐喻冷战（片中也提到了这部电影的原著小说），展现男主灵感枯竭后吸食毒品后瘾的呈现与轮回审判。不过与前者冷峻以个体遭遇表宿命和后者欲望迸发的强私密性不同，刚看本片时感觉影片是一种对天主教的解构和带有决定论逃避现实的感觉（这内容可能与导演的情况有关），作品的内容也非常像前作《坏中尉》，不过是把性，暴力和毒品等元素更加的抽象了出来，并用带有决定论角度去阐述（《坏》中男主的罪是自己犯的，也是遵循了传统天主教的形式即有罪之人的自我救赎，而刚开始我认为本片应该将遵循同样的逻辑），直到片中她遇到了她的Predecessor，我觉察到有丝丝不对，稍做回想后，我感觉影片并不仅仅是对于传统基督教的解构，而是针对杨森主义（亦或者加尔文主义）的一种另类诠释。以其部分观点作为一个根基展现了社会的状态，如果让那不可控制的欲望吞噬一切，即走向决定论，也就遂从了杨森主义。如恩典名字早已预定等。但接近结尾处对牧师的忏悔和结尾处的复活或是重生则像是走出了尼采自由意志或者是齐克果信仰飞跃的路（哲学答案）亦或者代赎的路（神学答案）。
宗教和哲学元素：
影片的社会状态整体呈现的是杨森主义和加尔文主义的Tulip，而女主就在此间走出了一条道路。
1. 强调原罪和全然堕落T（Total Depravity），即世间的人一出生皆有传承自亚当和夏娃偷食禁果之原罪，它是欲望也是瘾。因为原罪才会有犯罪，人本身就是罪恶，无法行善，所以感觉这是对影片中社会的呈现，整部影片都被大量的罪恶包裹着，但女主直面欲望和结尾的将之摆脱是由自由意志的呈现，即打破无法行善的桎梏。（遇到的Predecessor感觉又不完全相同，可能区别在于承认和否认）。
2. 无条件的筛选U（Unconditional Selection），这里也是我认为跟《坏中尉》最明显不同的地方，女主的启迪是一步一步进行的，从Predecessor的哲学思辨到年轻的传道人的选择再到结尾处对于原罪的认识。
3. 限定的救赎L（Limited Atonement），耶稣并不是对全人类的救赎，而是为选定好的人，这里也是决定论的部分呈现，而也是我认为导演带有部分自我逃避的感觉。
4. 不可抗拒的恩典I（Irresistible Grace），这里对影片有两部分呈现，首先是把极致的欲望投射给人是可以看做恩典，永生的生命（毒品是极致的快乐）。其次这可以解释开始的时候女主无法让拒绝Casanova和同样的后期其他人都无法拒绝女主。影片中唯一拒绝过女主的人是接近结尾处公寓门口的年轻传教士，他的拒绝也是导致女主回到公寓后发狂的原因，可能是作为堕落感召被拒绝后的副作用。
5. 圣徒的坚韧P（Perseverance of the Saints），被选中的人就会被拯救，而且也将永生。不过感觉以结果论是否在限定救赎总有些反常识，不过影片还是展现了即使是罪恶的体现，仍然可以得到耶稣的救赎。
画，影片中出现过很多画，第一次注射血液时女主看着的画是女子手拿荆棘之冠，开始我以传统天主教看是预示着女主注射血液为世人赎罪，同时也可能是象征了偷食禁果之罪早已传递给了世人。后来的感觉是欲望的放大，其实可以看出影片多次出现残破的战争图景就是欲望放大极致的结果。咖啡馆内《耶稣复活》的画作则早早的预示着女主复活的结局。
笛卡尔的我思故我在和萨特的存在先于本质是相悖的，而片中的社会呈现是支持萨特的观点，因为我罪故我在，欲望即是原罪在我降世时就已经存在，所以是存在先于本质的，即无法改变。然而结尾却恰恰证明了笛卡尔的观点，即另一套思想的重生。
抽血男教授那里能看到胳膊上有In和Out的文字，对应的应该是羔羊是否被拯救的Saved和Unsaved。
视觉上，整体的镜头都挺喜欢的，以手持跟镜头和POV呈现出两种视角效果很好，手持的抖动幅度还挺不错，还有几次的过肩镜头非常喜欢。蒙太奇剪辑出现的次数也很多，类似于泥土毒品的对照。黑白光影处理的很棒，尤其是血液在片中呈现出的黑色感觉可能比彩色电影呈现出的效果还要好，其实这种风格让我想到了68年的《活死人之夜》，呈现的效果差不多，而印象最深的几场光影是Casanova吸食女主的那场如囚笼般的视觉呈现，女主压抑不住欲望时的打光犹如《诺斯费拉图》中吸血鬼的影子。而影片开场的风格和内心独白的呈现则是让我想到了《很可能是魔鬼》。
听觉上，大量现场收音，不过部分配音部分音色差还是比较明显的。影片最频繁出现的配乐是Onyx的《Betta Off Dead》，还特意搜了一下这支首歌，主要体现是边缘化的社区面临的困难和挣扎。剩下的就是大量的沉重音效和一些鼓声，都是帮助烘托一种恐怖氛围。
不知道影片这种解读哲学是对纳粹发展史的讽刺，还有血液在影片中作为介质是否有艾滋病隐喻。
