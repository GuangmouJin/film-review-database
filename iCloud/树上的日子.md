《Des journées entières dans les arbres》（《整天在树上》）讲述的是一对母子之间复杂的情感关系。这部戏剧及电影关注的是人物的内心世界和他们之间的微妙互动。

### 故事梗概

故事的主角是一位年老的母亲和她的儿子。母亲多年未见儿子，于是前来探望他。儿子是一个生活放荡的人，他的生活充满了不安定与颓废，他与一个年轻的女人（可能是他的情人）同住。

在母亲短暂的探访期间，二人之间的交流显得极为疏离。母亲试图重新建立与儿子的亲密关系，并且有意无意地试图控制他的生活。然而，儿子对母亲的情感既有依赖又有抗拒，他试图维持自己的自由和独立性，但又无法完全摆脱母亲的影响。

在母亲眼中，儿子仍然是个需要照顾的小孩，而她则陷入了一种执念，似乎想通过掌控儿子的生活来填补自己内心的空虚。然而，这种执着反而加剧了他们之间的距离。儿子虽然顺从母亲，但内心却充满了逃避和抗拒。

### 主题与意义

杜拉斯通过这对母子的关系，探讨了亲情中的控制与依赖、孤独与爱等主题。母亲代表着过去和对传统家庭观念的执着，而儿子则象征着对自由的渴望和对旧有束缚的抗争。

整部作品充满了杜拉斯典型的孤独感和对人类情感复杂性的深刻洞察。故事的核心是人际关系中那种既想亲近又无法真正沟通的矛盾情感，这种情感在杜拉斯的作品中屡见不鲜。

最终，这部作品并没有为观众提供一个明确的结局，而是留下了开放性的思考空间，让人去反思亲情关系的本质及其背后的孤独感和无力感。



《Des journées entières dans les arbres》并没有原著小说，它是玛格丽特·杜拉斯的一部戏剧作品，而非小说。杜拉斯于1965年创作了这部戏剧，并在之后将其改编为电影。因此，尽管杜拉斯以小说创作为主，这部作品在她的创作中是以戏剧的形式出现的。


《Des journées entières dans les arbres》（《整天在树上》）是法国作家兼导演玛格丽特·杜拉斯（Marguerite Duras）1977年执导的电影。这部电影改编自她的同名戏剧作品，展现了杜拉斯独特的叙事风格和她对人性、孤独及情感关系的深刻探讨。

### 影评

#### 1. **叙事与风格**
杜拉斯的电影一向以其独特的叙事风格著称，她喜欢通过静态的镜头、缓慢的节奏以及极简的对话来展现人物内心的孤寂与复杂的情感世界。在《Des journées entières dans les arbres》中，这种风格得到了充分的体现。电影几乎没有传统意义上的剧情发展，而是更像是一系列充满情感张力的片段。这种非线性的叙事方式让观众感到似乎在观看一幅幅情感的画面，而不是在追踪一个故事的进展。

#### 2. **人物与情感**
电影中的人物关系充满了杜拉斯作品中常见的对人际关系的深刻审视。母亲与儿子之间的关系是影片的核心，他们之间的交流既亲密又疏离。母亲是一位孤独的女性，她试图通过控制儿子来填补内心的空虚，但她的这种情感需求却让人感到压抑和窒息。儿子在母亲面前表现得逆来顺受，但内心却渴望逃离。这种微妙而复杂的情感关系正是杜拉斯所擅长刻画的，电影通过细腻的表演和对话将这种情感展现得淋漓尽致。

#### 3. **视觉与空间**
杜拉斯对空间的运用极具特色。在电影中，室内空间既是人物的庇护所，也是他们的牢笼。镜头往往长时间停留在封闭的房间内，给人一种压迫感和窒息感。同时，杜拉斯对光影的运用也十分考究，昏暗的光线与人物内心的阴暗形成了呼应，营造出一种既真实又梦幻的氛围。

#### 4. **语言与文学性**
作为一名作家，杜拉斯对语言有着深刻的理解和敏锐的把握。影片中的对白极具文学性，简洁而富有哲理，给观众留下了深刻的印象。语言在这里不仅仅是交流的工具，更是表现人物内心世界的重要手段。杜拉斯的语言风格既充满诗意，又带有一种冷峻的现实感，让观众在聆听这些对话时感受到了一种独特的情感震撼。

### 总结
《Des journées entières dans les arbres》是一部典型的杜拉斯电影，充满了她对人性、孤独和情感关系的深刻思考。尽管这部电影的节奏缓慢，情节相对简单，但通过对人物内心世界的细腻刻画和独特的视觉风格，杜拉斯成功地创造了一种具有强烈情感张力的电影体验。这部电影虽然可能并不适合所有观众，但对于那些喜欢杜拉斯风格的人来说，它无疑是一部值得反复品味的佳作。
