# 杜拉斯短片1

杜拉斯短片1 
她在威尼斯时的名字在荒凉的加尔各答

#ICA 固定镜头，歌剧演唱，听不懂。

#ICA 同样的固定镜头，利用光线和焦距的调整来展现画面的不同层次。

#ICA 1976年的作品，晚于《印度之歌》，与《印度之歌》相同的采用了音画分离，并且音频完全源自于《印度之歌》的，感觉顺序都没有调整，而画面则99%的空镜，只有在结尾处有几个角色露面，全片采用静态镜头和缓慢的镜头移动，体现的仍然是孤独这个主题。影片似乎是另一段“酒店”（记忆）的呈现，荒废破败的感觉似乎对应了殖民主义的消亡亦或者那无人守护的记忆。

最美妙的在于我昨天刚看完《印度之歌》，所以对于同样的旁白的印象还是蛮深的，同样的台词就导致很多时候我似乎能“看到”《印度之歌》的画面，这就仿佛是两段“画面记忆”的交织。

视觉上，空镜的效果非常好，非常的冷酷，少有几处手电的光亮视角能感觉出一丝温暖，感觉情感和叙事的高度分离，只有结尾处角色出现时的灯光给画面带来一丝温度，仿佛置身于一个时空停滞的梦境之中。



### 影评

《Son nom de Venise dans Calcutta désert》是一部充满诗意且令人沉思的电影，它挑战了传统叙事的界限，展现了杜拉斯对电影媒介的独特视角。杜拉斯通过静态镜头和缓慢的镜头移动，营造出一种静谧而又神秘的氛围，

电影的背景设定在一个荒废的殖民地庄园，这一地点本身就充满了象征意义，暗示着逝去的时代和消逝的记忆。杜拉斯使用庄园的废墟和荒凉的景象，隐喻着人物内心的孤独与绝望。观众在观看电影过程中，仿佛可以感受到时间在这些废墟中缓慢流逝，逐渐侵蚀着一切。

与《印度之歌》类似，这部电影的声音设计极具特色。电影中充满了回响的声音和遥远的对白，这些声音像幽灵一样在废墟中回荡，给人一种梦幻般的感觉。杜拉斯通过声音的运用，进一步模糊了现实与幻觉的界限，使观众在声音和影像的交织中，体验到一种超现实的氛围。

电影中的人物并没有传统意义上的具体形象，他们更像是一些漂浮在记忆中的幻影。杜拉斯通过这种非具体化的人物形象，探讨了身份和存在的模糊性。观众无法确切地了解这些人物的背景或动机，但正是这种模糊性，增强了电影的神秘感和情感的深度。

总的来说，《Son nom de Venise dans Calcutta désert》是一部极具实验性的艺术电影，它通过独特的视觉和声音语言，探讨了深刻的情感和哲学主题。杜拉斯在这部作品中展现了她作为一名导演的大胆创新和非凡才华。虽然这部电影对于一些观众来说可能过于晦涩和难解，但对于那些愿意沉浸其中、细细品味的人来说，它无疑是一部值得深思和回味的杰作。
