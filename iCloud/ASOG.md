#BFI #LFF #Laugh 长评-Grief changes shape, but it never ends。以喜剧形式呈现悲剧故事，喜剧效果在其中是没有冲淡悲剧色彩的，非常厉害。整个故事可以看作是一场公路电影，以男女主旅程带观众看天灾台风的影响，但其中不光是天灾，也夹杂着人祸，中途遇到的人事物让观众可以从侧面了解事件影响。影片中最成功的应该是角色，尤其是女主，她的身份没有可以被特意挑出来表达，她的坚强和乐观让故事上就有一个升华，而民族性层面也从片中一个个得到一个很好的呈现。视觉上非常丰富，动画，异象，吊线玩偶，大远景，空镜，航拍。《黑客帝国》墨菲斯强势插入，还有基努的金句，Grief changes shape, but it never ends。导演Intro时说他去到菲律宾在台风受难纪念日时问是否有喜剧，本来以为对方会觉得冒犯，结果对方告诉他确实有，也是由此认识的女主，喜剧表演结合唱歌跳舞的派对形式也是很独特。不喜欢的点可能就是结尾处的悲伤分段爆发的形式比较差，感觉堆积到最后一次次泄气了。
