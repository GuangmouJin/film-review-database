# 伦敦｜【画展推荐】培根肖像画作品展

# 


弗朗西斯·培根（Francis Bacon, 1909-1992）是一位英国画家，以他独特的扭曲，痛苦和强烈情感的肖像作品而闻名。他的艺术风格结合了表现主义和超现实主义，探索了人类的孤独、恐惧和内心挣扎等主题，呈现出对人性的深刻而痛苦的理解。

其实今年的年初Tate和前年RA都有他作品的策展，尤其是前年RA策展了他大量的三联画创作，这次的策展主要是侧重肖像艺术，不光包括他对早期其他艺术家肖像的回应也也包括他的自画像和其他周围朋友和恋人的画像和习作。

地点：National Portrait Gallery
时间：至25年1月19日
价格：成人23胖；学生27胖；MA卡免费

其实非常推荐喜欢逛展的小伙伴办一张MA（Museums Association）的卡，像Tate和本展都可以免费线上预订，还有大量的合作展馆有打折。
