# 爱情短片

# 



#CineLumiere 还蛮喜欢影片情绪处理的，能看到类似《后窗》《偷窥狂》的设计，不过剥离了前者的悬疑惊悚和后者的暴力死亡，回归到对情感与爱情的解构，权力在窥探和被窥探者之间流转处理的非常，不过结尾处的处理还是太过理想与美好，对照看《对她说》那种偏向于批判的现实主义处理我更喜欢一些。爱在影片中就像齐泽克的理解般是暴力的，男主无法继续观看女主跟别人一起和他迫切的寻求接触都是占有欲的表现，牛奶洒倒后他的介入都是一种对牛奶纯洁性的讽刺，所以男主这个角色其实更像是一个幻想破灭的未来。男女主在某种程度是存在对照性的，他们都是用自己的形式去拆解爱情，女主是用身体，男主则是用幻想。这么一看结尾处女主的处理其实也不光是源于愧疚，还带有着对于曾经的缅怀（澳洲的信件）和未来的期许（最后呈现出的幻想）。

其实很喜欢女主作为被窥探者权力调转的处理，很像是BDSM中的权力结构塑造，但调动权力后触发男主崩溃后的“观看”，这部分力很难剥离愧疚，我觉得这部分很不喜欢，一方面是对于男性脆弱的包容，另一方面是对于爱情这个幻想的肯定，爱在影片中作为主线的表达是被模糊了的，而且女主的观看在这部分也是同样弱化的，原本可能呈现的反凝视也并不能成立，她的观看不再具有重构关系的力量，而是被整合进对男性创伤的理解之中，从而使权力的流转再次中断。

视听上，镜头和构图上效果非常好，大远景和固定镜头都是在增加人与人的距离感，同样影片中的台词也很少，都是描述社会的状态。这部作品人与人之间的关系的塑造让我想到了**夏娜莱克**的作品风格，那种触碰能力丧失，语言的失效和欲望被压抑的状态。非常喜欢那种偷窥场景的塑造，带有着元电影的质感，观众本身也是在透过那个望远镜重复男主的行为。片中色彩的呈现也很有意思，透过那个偷窥镜头仿佛就能看到女主家的蓝白红配置，还有大量的三色元素，红色本身就与欲望相关联，不论是女主的床，还是他俩对视的那场戏的红色背景都带有极强的性隐喻。最喜欢男主坦白的那场戏，坦白被骂后紧接着就是一个远景低机位前景遮挡，表现出的是女主当下的不知所措，但紧接着那个背景又似乎带出了她的回忆。配乐部分比较单调，后配音统一性强，间离感也强。环境音最为突出，非常喜欢，尤其是两人打电话的设计。台词不多，是非常喜欢的效果，那种透过表演，表情特写呈现的心理比台词更有力量。
