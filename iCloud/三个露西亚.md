# 三个露西亚

<u>**Lucí**</u>a




#ICA 长评-“露西亚”的困境。以三个时期三个阶级同名为露西亚的三位女性的三个独立故事构成，故事独立，但是面临的困境类似，爱情元素更多的是以幻觉出现，女性的爱与付出在“大是大非”面前被消解，带反男性英雄表达。1895年殖民时期西班牙统治下的古巴，上层女主面对的抉择参与更宏大的叙事，浪漫的主体在历史洪流中微不足道，苏联蒙太奇和德国表现主义风格呈现被环境压抑的情绪。第二个故事是1932年读材时期，中产女主以回忆的形式呈现了那迷茫的时期，结尾处的死亡呈现的是意识觉醒的代价，个体消失，回忆更贴近于心理表达，空间与角色的关系想到了安东尼奥尼。第三个故事是196…年歌名胜利之后，强调的都是革命话语下的解放是理想，现实层面底层的女主则仍被父权结构束缚，新现实主义结合喜剧表达，欢笑与苦痛交织更显真实。




影片以三个时期三个阶级同名为露西亚的女性的三个独立故事构成，感觉每个故事内容上独立，但是她们都面临着类似的困境，爱情这个元素在片中更多的是以幻觉出现，女性的爱深沉和付出但在“大是大非”面前被消解，感觉这也带有着一种反男性英雄式的表达。三个故事只有结尾的那个还算是有些希望，前面的则更多的是苦与泪。

角色和隐喻上，三个故事的主题似乎都是歌名时期女性主体的呈现，民族爱情，一战再战和个体“解放”，三个状态都挺不错的，作为共同体呈现了女性共同体困境的循环，而且每个时间节点都挺关键的。第一个故事是1895年殖民时期西班牙统治下的古巴，这个故事的篇幅最长，露西亚作为上层女性面对的抉择更加的宏大，但男女主之间的关系带有着阶级和意识形态的斗争，观众似乎也从这个角度看到浪漫的主体是如何被历史洪流冲走的。第二个故事是1932年读材时期，中产女主露西亚以回忆的形式呈现了那热血又迷茫的时期，结尾处的死亡呈现的是意识觉醒的代价，似乎也是容易被忽视的个体的处境。第三个故事是196…年歌名胜利之后，底层女性的状态这部分也让我想到了头几天看的《直至某刻》，强调的都是革命话语下的解放是理想，现实的露西亚则仍被父权结构束缚。

视听上，三个故事视听风格各不相同，第一段明显带有苏联蒙太奇和德国表现主义风格，故事的平行剪辑是一种隐喻，高对比黑白则是一种压抑情绪，手持和快速剪辑都是极度情绪化，还有那种脸部特写看着也是扭曲的不行。第二段则是以回忆呈现，更贴近于心理表达，空间虽然更加的现代，但感觉角色间的距离却非常有距离，想到了安东尼奥尼…镜头相对稳定，但以内心独白结合的固定镜头给人的感觉还是不安，那段癫狂宴会似乎是毁灭的原因。第三段则是新现实主义结合喜剧表达，长镜头蛮多的，感觉时空更加稳定，但那就如同电影中的表象呈现，物理和心理层面看则更加受困，那种欢笑与苦痛交织更显真实。片中配乐更贴近古巴音乐的风格，台词则是以叙事和讽刺为主，感觉也是看到希望的原因，毕竟整体风格还是偏向轻快的。



	•	第一段：封闭庄园 / 上层空间 → **阶级与殖民结构**
	•	第二段：城市与室内空间 → **现代性与压抑关系**
	•	第三段：乡村与开放空间 → **革命后的“自由幻觉”**

但注意：

👉 空间越开放，控制并未消失，只是转化为日常权力
