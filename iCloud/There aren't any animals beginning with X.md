# There aren't any animals beginning with X

BFI Frames of Mind: The Films of Peter Greenaway 整个故事完成度非常高，死亡也贯穿整部电影，这感觉跟导演的人生观有很大关系（他声称自己会在80岁结束自己的生命，正巧今年他80岁），影片以死亡开场，到腐烂的苹果和各种动物，最后到三人的死亡结束，带有很强“natural history” 博物学（这也是他自己在访谈中提及过的）的感觉，把观察，记录，描述进行到了一种极致，甚至于整部电影都是我们对于ZOO的观察（这里很明显，因为女主已经只是镜头来问观众了），直至他们死去。
视觉上，大量的固定镜头，用于展示那极至的对称构图（刚看完《自由的幻影》的我脑海中不禁想到了那句对于对称的抱怨），还有后移从而展现给观众更广的空间，带有一种悬疑感。最后半环绕镜头更是带我们悄悄走近他们，近距离观察他们（最后的消毒。光线阴影的运用也是很棒，红蓝光线切换的诡异，红光的欲望，被门“一分为二”的影子等都很不错，最喜欢的一段是对于情绪迸发时的辅助作用，印象是想要孩子的那场戏，情绪爆发前屏幕突然变暗，然后一声怒喝，紧接着光线进入，人物也和颜悦色了起来，非常震撼。
听觉上，主要事迹受配乐循环使用，《time lapse》非常强的递进节奏强度，能很好的传递渐强的情绪波动，《lady in the red hat》每次响起都让我想起《dance monkey》的前奏，这个曲子似乎对应着影片中跟“医生”一直在一起的人，非现实感很强，另一首配乐则是出现在快进的腐烂过程中，这里总让我想起《发条橙》中《威廉退尔序曲》那段儿“表演”，而且我感觉影片内核上在某种方面跟《雏菊》和《发条橙》很像。
导演自述，Z是字母表最后一位，也是Z is for Zebra，斑马作为黑白动物在导演看来仍然未被解答是黑底白纹或者白底黑纹（斑马是黑底白纹，看《我出生了，但……》时特意查询了下这个知识点），而且斑马是登上诺亚方舟的动物，充满着神秘色彩。导演还Q到了柯南伯格…咳咳具体说了啥…就不提了。
细节，报纸上我好像看到了在《格林纳威字母谈》中被溺死在浴缸中的“人”的报道，感觉剩下的几个标题应该也跟他其他作品可能有呼应。
