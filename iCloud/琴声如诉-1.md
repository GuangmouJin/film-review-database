# 琴声如诉

# 
#ICA 影片改编自杜拉斯本人的小说，故事对照小说改变还是挺大的，其中主要是风格，小说中的意识流是非常明显的，而电影更加贴近于现实主义。电影的名字在片中展现得更加清晰，亦为女主的感情状态，最后结尾处她的做法跟她儿子其实是一样的。讨论的话题对照咖啡馆内的情侣没有小说模糊，更加直接。

角色上，改编的也挺多的，尤其是女主这个角色设定，原作中感觉更加有自主性，并且在结尾处也不是这样，不知道因为什么原因做了调整（Intro说这部小说跟杜拉斯自己的经历有很多相似之处），男主在片中也更加“现实“了，装逼的质感不是很喜欢。阶级的窒息感塑造的挺好的，雷同教小孩学琴的老师。

视觉上，镜头还是蛮喜欢的，尤其是几处推轨侧跟的镜头，只是剪辑如此“落地”的故事有些可惜（非常期待是一个类似于《去年在马里昂巴德》的视觉呈现，因为原作中对于现在的表达是非常模糊的），还有两处跳剪衔接的有问题。构图很不错，那海边的长椅上的背影和后续的呼应很美，还有大量空镜展现的现实。

听觉上，主要是小男孩的钢琴声，也是表现女主的状态，有几处后配音音色差明显。


《琴声如诉》（Moderato Cantabile，1960）是一部由彼得·布鲁克执导，改编自玛格丽特·杜拉斯同名小说的电影。影片由让娜·莫罗和让-保罗·贝尔蒙多主演。以下是这部电影的剧情概述：

### 剧情概述

#### 主要角色
- **安娜·德巴**（由让娜·莫罗饰演）：一个富有但感到孤独的工厂主妻子。
- **肖法尔**（由让-保罗·贝尔蒙多饰演）：一个工人，后来成为安娜的朋友和倾诉对象。

#### 故事情节

1. **开场与谋杀**
   - 故事开始于一个小镇的咖啡馆。安娜在她儿子的钢琴课上听到一声枪响。她和其他人一起跑出去，发现一个女人被谋杀。这个事件成为影片的催化剂，唤醒了安娜内心深处的情感和欲望。

2. **安娜的困惑**
   - 安娜是一个被生活和婚姻束缚的女人。她在钢琴课上学到的音乐术语“Moderato Cantabile”（中板，歌唱般的）成为她情感状态的隐喻。她开始对谋杀事件和死者的生活产生强烈的兴趣。

3. **与肖法尔的相遇**
   - 安娜在咖啡馆遇见了肖法尔，一个普通的工人。他们之间开始了一段复杂而模糊的关系。安娜向肖法尔倾诉自己的孤独和对生活的困惑。两人之间的对话充满了暗示和未说出口的情感。

4. **探索与迷失**
   - 随着时间的推移，安娜和肖法尔之间的交流越来越深。他们在咖啡馆的对话成为影片的核心，探讨了孤独、欲望和人类关系的复杂性。安娜试图通过这段关系寻找自己内心的答案，但却发现自己越来越迷失。

5. **开放的结局**
   - 影片以一种开放式的结局结束，安娜最终没有找到明确的答案或解决办法。她的生活和内心依然充满了困惑和不确定性。

### 主题与风格

《琴声如诉》是一部充满内省和哲理的电影，通过细腻的情感表达和深刻的对话探讨了人类的孤独和欲望。杜拉斯的小说和布鲁克的电影都以其简约而富有力量的叙事风格著称，使得这部电影成为一部经典的艺术作品。

### 总结

《琴声如诉》不仅是一部关于爱情和孤独的电影，更是一部探讨人类情感深层次的作品。通过安娜和肖法尔的复杂关系，影片揭示了人在面对生活的困惑和内心的孤独时所经历的情感挣扎。影片的开放式结局也给观众留下了深刻的思考空间。
