# 4. 《终》（End / Verj, 1992, 10分钟）

# 



在上，佩列申拍摄了不同年龄与族群的男女；火车成为人类的缩影，而这段旅程成为生命的隐喻。

佩列申将《终》视为《开端》的“对应作品”。

#ICA 比较喜欢的一部，火车上的社会缩影，影片拍摄于20世纪90年代初苏联解体之际，在从莫斯科驶向埃里温的火车上拍摄，感觉火车一直是充当历史车轮的隐喻，在影片中乘客更接近于社会的大多数，不同的年龄和族群的人们呈现的是人类共同体的象征，旅程感觉还是有着积极的表达，也是苏联解体时期的一宗思考。蛮喜欢长焦特写镜头的呈现的，凝视更多的是内心的外化。


**3）象征 / 隐喻**

* 火车 → 历史进程 / 命运轨道
* 车厢 → 社会结构
* 乘客 → 人类整体
* 旅程 → 人生

👉从民族 → 人类整体

⸻

**4）时代背景**

* 苏联解体
* 身份、秩序崩塌

👉影片成为一种“过渡时代的冥想”

⸻

**5）视听语言风格**

* 更平静、内省
* 面孔特写成为核心
* 节奏趋缓（接近沉思状态）

👉从“冲击”转向“凝视”

⸻

**6）风格来源 / 影响**

* 回归早期主题（呼应《开端》）
* 影响后来的“essay film”

👉是佩列申晚期的哲学总结

⸻

最后给你一个结构提示（方便你写影评）

如果你之后要写，可以用这条演进线：

《我们》（民族历史）
→ 《居民》（人类批判）
→ 《四季》（自然宇宙）
→ 《终》（人类整体命运）

👉这其实是一个不断“去中心化”的过程。
