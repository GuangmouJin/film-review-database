# 悬崖上的野餐


欲望之下皆蝼蚁

#PCC #TheFilmsofPeterWeir 神秘主义作品，从片中看到了很多作品的影子，如《黑水仙》和《豹族》的那种性压抑的环境和《第七封印》手牵手带有极强存在主义的舞蹈。片中出现的很多种动物隐喻，尤其是蚂蚁，蜥蜴和天鹅，蜥蜴作为四脚蛇的暗示极为明显，而天鹅洁白无暇几次出现的节点也都是在映射Miranda。

影片跟欲望有极大的关系，但并不是异性之爱。开场没多久McCraw就在车中对悬崖成型原因进行了描述，其中带有极强的性隐喻以至于她完全沉浸其中（镜头非常能明显）最后忽然回神，其实基本就可以确定她在悬崖中体验过性爱并且心神向往，这在跑回的女生嘴里说出过她看到了McCraw只穿了一个内裤，并且强调了红云，红色在片中是极少出现的，第一次出现是McCraw的衣服颜色，另一次出现则是被拯救的Irma再次回到同学间时穿的衣服的颜色，可以推测红色和某种体验有关，并且并不是与男性的性行为，所以合理推测的话是女性同性行为，这么看也可以倒退Sara和Miranda在开场时的关系，并不是一个人的两面设定，而是同性恋人。而老师对Miranda的描述为Bottucelli angel，画作中则是《金星的诞生》中的维纳斯，值得注意的是维纳斯在画面中也是删减版，并没有展现全画的裸体，感觉意为同样的压抑。最后的是Bertie手上那裸女纹身与他对Irma的“拯救”感觉与男主的天鹅执着都太过于刻意表现对于女性之美的欣赏。他俩之间在我看来反而却有些说不清道不明之感。

视觉上则是很契合影片风格，远景柔焦的迷幻感，近乎垂直仰拍的诡异，悬崖石头的特写的压迫感，炙热下那扭曲的画面的情欲，双爆的隐喻和推焦的角色塑造都非常舒适。最后结尾处的抽帧处理也非常符合回忆的美妙。

听觉上，片中很多配乐感觉是使用的八尺，时而清脆阳光表现的一股隐隐的情欲，时而悠扬透露出丝丝的诡谲。还有舒缓的钢琴曲搭配着管风琴宗教感也非常强。不论是踩过草地的声音还是蝉鸣都太棒了，
