# 《路易十四的崛起》：权力的仪式化与影像的去戏剧化

# 



#CineLumiere 罗西里尼的这部作品给我的感觉就是克制，不光角色设计的克制，整体的表达也相当克制，以平铺的讲一个“凶险”的故事，作为一部传记片这种去戏剧化的处理还是很大胆的。电影的结尾设计尤其好，当一切结束，路易十四自己脱掉了那厚重的衣服，我们也随着他松了一口气，视听上并没有奢华宏大的场面，而是采用了日常化的吃穿住行来展现角色的权力，也算非常完美的呈现了角色那独特魅力，镜头使用的也蛮好，尤其是摇镜头的使用，结合整个环境的特点很符合布列松的那种极简主义风格，一切像是建立在在无数次精心编排。影片的用声极为克制，台词基本也是平铺直叙，环境音的效果与视觉的风格呈现是相辅相成。



《路易十四的崛起》（La prise de pouvoir par Louis XIV, 1966）影评

罗西里尼的**《路易十四的崛起》是一部极具克制和观察性的历史电影，它没有宫廷剧惯有的戏剧化冲突，也没有英雄主义的神话塑造，而是用近乎冷静的现实主义手法**，记录了绝对权力的构建过程。这不仅是一场政治的演出，更是一场身体、服饰与权力的演变，罗西里尼的镜头以一种近乎文献式的方式，呈现了一个年轻国王如何通过符号、仪式和策略，完成个人权力的塑造。

1. 权力的构建：符号与仪式的统治

影片没有直接展现战争、宫廷阴谋或宏大的政治斗争，而是通过一系列细节，展示了路易十四如何在象征层面上确立自己的统治。从他改变服饰，到强化宫廷礼仪，再到让贵族依赖宫廷生活，这些看似微不足道的决定，实际上构成了君主集权化的核心策略。
	•	服饰的政治性：电影最具代表性的场景之一是路易十四决定穿上更加华丽、繁复的服装。这并非只是个人审美，而是以视觉形象强化权威的一种手段。通过服饰，他不仅在外表上塑造了“太阳王”的形象，更让贵族们被迫效仿，形成了一种以奢华为控制手段的政治体系。这一点让我想到《坏时机》中席勒的画作，身体的扭曲不仅是艺术的表达，也是权力关系的隐喻。
	•	日常生活的戏剧化：影片中的宫廷生活并非自然的，而是一场经过精心编排的舞台剧。路易十四的饮食、起居、社交，都被塑造成一种必须被观看的行为。王权不再依赖于军事力量，而是通过仪式感和视觉化的控制，让所有人意识到君主的权威无所不在。这与帕索里尼在《索多玛120天》中对权力与秩序的描绘不同，帕索里尼通过极端的身体剥夺展现暴力，而罗西里尼则通过繁琐的仪式展现权力如何温和地渗透进社会结构。

2. 现实主义的历史影像

罗西里尼的历史电影有别于传统的史诗片，他不追求戏剧化的情节，而是以极端克制的影像语言，让观众沉浸在权力运行的日常之中。《路易十四的崛起》中，固定镜头和缓慢的长镜头被大量使用，使电影更像是一部关于政治行为的观察性纪录片。
	•	长时间的静观：罗西里尼几乎不使用传统的蒙太奇，而是让镜头长时间停留在王室仪式、谈话、甚至静态的空间之上。这种方式让观众直面历史，而非通过剪辑塑造的情感引导去理解它。
	•	去戏剧化表演：演员的表演方式刻意去情绪化，没有夸张的情绪波动，路易十四的决策不是以戏剧性的台词呈现，而是通过一种平静、缓慢但坚定的方式展开。这种表演风格接近布列松的“模型”理论，让权力的构建显得更加客观，而非基于个人英雄主义。
	•	空间与权力的关系：宫廷的空间被拍摄得极为封闭，贵族们围绕着国王转动，而国王始终处于视觉中心。这种调度方式强化了权力在空间中的展现，也让人联想到库布里克在《巴里·林登》中对贵族社会的观察——在优雅的秩序之下，个体的自由被削弱，成为被权力体系框定的符号。

3. 绝对权力的诞生：从人到神的转变

影片的核心并非简单地展现路易十四如何掌权，而是探讨权力如何从现实层面转变为神话。年轻的路易十四在影片开始时并未完全掌控局势，他必须逐步构建自己的形象，直到最终，他不再是一个普通的人，而是成为了**“太阳王”这一概念本身**。
	•	从个人到符号：电影的结尾，路易十四的形象已经完全变成了一种视觉化的权力符号，他的存在不再是个体，而是一个超越现实的象征。这种转变让我想到法斯宾德的《玛丽娅·布劳恩的婚姻》，玛丽娅同样从一个现实中的女性，最终被塑造成一个政治与社会变迁的象征性形象。
	•	视觉的操控与传播：影片中反复出现的贵族围绕国王的镜头，不仅仅是封建社会等级制的体现，更是权力如何通过被观看来维持自身的例证。国王必须不断制造被观看的场面，才能持续巩固自己的统治。这种权力运作方式，让我想到《大都会》（Megalopolis）中对视觉奇观的批判，权力不仅仅是法律和军队，它更是一种影像、一种仪式、一种无法逃离的观看关系。

结论：冷静的权力解剖

《路易十四的崛起》是一部去除了戏剧性、回归历史本质的电影，它用克制的镜头、精准的仪式感和符号化的视觉语言，呈现了一位君主如何通过形象与仪式构建绝对权力的过程。罗西里尼没有美化王权，也没有赋予人物英雄主义色彩，而是将一切简化为一场关于视觉、空间和符号的政治实验。

这部电影并不像传统历史片那样试图还原“伟大事件”，而是以一种观察性的冷静态度，让观众直接进入权力运作的核心。它既是关于历史的影片，也是关于电影如何再现历史的实验。在它精准、缓慢、充满象征意味的影像中，权力本身成为了唯一的主角，而路易十四，只是一个借由影像诞生的神话。



路易十四的历史背景

路易十四（Louis XIV，1638-1715），即法国波旁王朝的国王，1643年至1715年在位，是欧洲历史上统治时间最长的君主之一（72年）。他被称为**“太阳王”（Le Roi Soleil），以其推行绝对君主制、中央集权、文化艺术繁荣**等政策闻名，奠定了法国君主专制的巅峰，同时也为后来的法国革命埋下了伏笔。

1. 童年与权力斗争
	•	1638年：路易十四出生于法国国王路易十三和王后安娜·奥地利之子，直到他四岁时（1643年），路易十三去世，他正式继位，由母亲安娜·奥地利摄政，并由意大利枢机主教**马萨林（Mazarin）**担任首席大臣，实质掌权。
	•	1648-1653年：在他年幼时，法国爆发了投石党之乱（La Fronde），这是一场由贵族和部分议会成员反对王权的叛乱，导致年轻的路易十四与母亲被迫逃离巴黎。这场经历对他影响深远，使他意识到贵族的威胁，并坚定了日后要强化中央集权、压制贵族的决心。

2. 亲政与绝对君主制的确立（1661年）

	•	1661年，在首席大臣马萨林去世后，22岁的路易十四宣布不再任命首席大臣，亲自掌权，这是法国历史上首次由国王直接执政，不受大臣控制的情况，标志着法国绝对君主制的正式确立。
	•	他采取了一系列措施巩固王权：
	•	削弱贵族：要求贵族们迁居到凡尔赛宫，使他们依赖宫廷的荣誉和国王的恩宠，而不是地方的军事实力。
	•	强化中央集权：削弱旧封建领主的权力，加强政府对地方的直接控制。
	•	重组财政：由财政大臣柯尔贝尔（Colbert）主导经济改革，推行重商主义，发展工商业，增强国家财政实力。

3. 文化与凡尔赛体系
	•	凡尔赛宫的建立：1661年起，路易十四开始扩建凡尔赛宫（Versailles），并于1682年正式将王宫迁至凡尔赛，使其成为欧洲绝对君主制的象征。
	•	文化的繁荣：
	•	支持艺术家、作家和建筑师，如莫里哀（Molière）、拉辛（Racine）、**勒布伦（Le Brun）**等，使法国成为欧洲的文化中心。
	•	建立皇家学院，如法兰西学院（Académie Française），巩固法语的地位。
	•	以自己为太阳神阿波罗的化身，推行宫廷礼仪制度，使整个宫廷成为精心编排的权力仪式。

4. 对外战争与霸权扩张

路易十四统治时期，法国成为欧洲最强大的国家，但他的扩张政策也带来了长期的战争和财政负担。
	•	荷法战争（1672-1678年）：攻击荷兰共和国，尽管取得部分领土，但未能彻底击败荷兰。
	•	大同盟战争（1688-1697年）：与欧洲多个国家（英、奥、西、荷等）爆发战争，最终在瑞士签订《里斯维克条约》，法国放弃部分占领地。
	•	西班牙王位继承战争（1701-1714年）：路易十四试图让自己的孙子腓力五世继承西班牙王位，导致欧洲大国联合对抗法国。尽管最终腓力五世成功成为西班牙国王，但法国国力已被严重消耗，国库亏空，社会不满加剧。

5. 晚年与死亡（1715年）
	•	1715年9月1日，路易十四在凡尔赛宫去世，享年77岁。他的曾孙路易十五继位（当时仅5岁）。
	•	路易十四去世时，法国财政危机严重，社会矛盾激化，虽然他的统治建立了法国的文化、军事和外交强权，但也为18世纪的动荡埋下了隐患。

结论

路易十四的统治是欧洲绝对君主制的巅峰，他通过仪式、建筑、文化和战争，塑造了一个**“太阳王”的形象，让法国成为欧洲的中心。但他的统治也过度依赖奢华的宫廷体系和战争开支**，最终使法国陷入财政困境，导致后来的社会危机和革命趋势。

在电影《路易十四的崛起》中，罗西里尼用极端现实主义的方式，冷静地展现了权力如何被精心制造、如何通过符号化的政治策略建立统治，而非依靠戏剧性的战争或阴谋。电影的影像风格正是对历史事实的一种解剖，让观众直观感受到绝对权力的构建并非一夜之间，而是通过日常细节、观看关系和精确的政治仪式一点点实现的。
