# 恒河女


#ICA 杜拉斯电影永恒的主题似乎都是孤独，影片展现了创作的过程，可以说这部影片就是《恒河女》的创作过程的模拟，呈现出的质感非常好，透过装满酒的酒杯看到的世界印象很深，大远景的孤独，尖锐的音效，冷静的旁白，虚焦的脸庞，手持镜头的晃动，还有那一望无际黑白波动的海和转身走进地狱般黑夜的镜头。



《甘格女人》是玛格丽特·杜拉斯“印度周期”的第一部电影:她在1964年至1971年间写了三本书，之后又拍了三部电影(分别是《印度之歌》和《印度之子》和《加尔各答之子》)。但是，正如灰色的天空和她的角色的毛皮外套所证明的那样，这部电影是在诺曼底特鲁维尔的寒冷气候中拍摄的:杜拉斯经常逃到那里写作(她晚年的许多电影都是在那里拍摄的)。

S. Thala几乎没有生命迹象。两个男人和两个女人，据说已经失去了他们的记忆，沿着广阔的沙滩漂流，看着一个旅行者(杜拉斯的前夫，迪奥尼斯·马斯科洛)到达(或者更确切地说，返回)。我们从两个没有肉体的女人的声音(包括弗朗索瓦丝·勒布伦的声音)中得知，他来到这里是为了结束自己的生命，几十年前他和一个已婚女人的婚外情一直困扰着他。但是，在一个空荡荡的大酒店里，这个女人又像幽灵一样出现了。

La Femme du Gange是对爱情、记忆和欲望永恒的探索。但不只是以旅行者和他爱的对象的名义。事实上，杜拉斯的意图是制作一部电影，在这部电影中，“过去，从个人传记的意外中解脱出来，没有混乱”将“同时是私人的和集体的”。

由本季策展人Daniella Shreir作简短介绍。



#ICA 复杂的情感，流逝的时间和消失的记忆。影片没有过分的流露情感，但却用角色关系和目的展现了那潜藏的情感。影片同样没有时间的概念，但却利用台词展现了时间的流逝。影片同有标明何处是记忆，但却派了人来守护。最喜欢的可能就是整体的风格呈现，电影也在开场明确了声音与画面分别是两部电影，而声音也不来源于影像中的角色，影片中我们随着一个男人走入触及他回忆的地方开始，空间与时间都消失了，也就是把记忆都融了进去似乎让一切情感都那么的遥不可及。白日与黑夜的轮替，前仆后继的海浪，无限旋转的转门，反复走过的酒店过道，最后都汇聚到了那个严肃的哲学问题上。

角色上，几个角色可以看作是不同时空下的记忆呈现，也可能是本身丢失了记忆的幽魂。黑衣的女主，她是唯一在男主那座记忆酒店拥有房间的人，其他的记忆就好似现实所见那般游荡在海滩。黑衣女主同样也是现实曾经的存在，而那张照片及游荡在酒店外的女人似乎是那内心想遗忘的情绪，即使进入了酒店也没有属于她的房间（她也出现在了《加尔各答的黑夜》中）。

视觉上，最明显的就是影片的镜头使用，电影只是用了固定镜头，其中只有小幅度的晃动，这种镜头太冷静了，就如同一个冷眼旁观的人，不过结合剧情看，感觉镜头的风格很大程度的保证了记忆的某种客观性。其次明显的就是景别的使用，大量的大远景和远景，全景，近景和中近景都非常的好，更不用提特写镜头了，整体呈现出的效果就是孤寂。最后的特点是重复，那些人就如同想让你记住他们般的重复走过那些曾经走过的路，也仿佛他们这么重复就不会消逝在男主的记忆中了。

听觉上，两个旁白似乎如同观察者般以另外两个视角讲述了一个故事，内容的重复性跟视觉的重复作用似乎相同，大量的自然环境音的呈现可能并不是现场收音，唯一一处渐强的处理让情绪可以更好的突出出来。


杜拉斯的小说《恒河女》（原名《拉霍尔之旅》，法文原名《Le Vice-Consul》）是一部描写印度背景的作品。小说主要围绕着法国领事和一名神秘女子的故事展开，通过多重视角讲述了几个人物之间复杂的情感纠葛和内心世界。

故事主要情节包括：

1. **法国领事的经历**：法国副领事被调到加尔各答，在那里他因无法控制的情绪和行为被逐渐边缘化。
2. **神秘女子的命运**：一位被社会抛弃的女子沿着恒河一路漂泊，经历了极端的贫困和孤独。她的形象象征着被忽视和压迫的女性。

3. **多重视角叙述**：小说通过不同人物的视角，展示了他们对恒河女的不同理解和反应，从而揭示了更深层次的文化冲突和个人内心的挣扎。

杜拉斯通过这些情节探讨了孤独、痛苦和人类情感的复杂性，描绘了一个充满矛盾和张力的印度社会背景下的故事。小说以其独特的叙事方式和深刻的心理描写而闻名，是杜拉斯重要的作品之一。


杜拉斯的小说《恒河女》和普契尼的歌剧《蝴蝶夫人》之间确实有一些呼应之处，尽管它们在情节和背景上有所不同。以下是一些主要的呼应点：

1. **文化冲突**：
   - **《恒河女》**：小说中描写了法国副领事在印度的经历，涉及东西方文化的冲突和不理解。
   - **《蝴蝶夫人》**：歌剧讲述了日本艺妓与美国海军军官之间的悲剧爱情故事，同样探讨了东西方文化的冲突和误解。

2. **女性命运**：
   - **《恒河女》**：恒河女的形象象征着被社会抛弃和忽视的女性，展现了她们的孤独和痛苦。
   - **《蝴蝶夫人》**：女主角蝶衣被军官抛弃，最终选择自杀，表现了她在爱情和社会中的绝望处境。

3. **孤独与边缘化**：
   - **《恒河女》**：法国副领事的边缘化和恒河女的流浪生涯都反映了人物的孤独和与社会的疏离。
   - **《蝴蝶夫人》**：蝶衣在被抛弃后陷入孤独，与社会和文化格格不入。

4. **情感纠葛**：
   - **《恒河女》**：小说中的多重视角揭示了人物复杂的情感纠葛和内心挣扎。
   - **《蝴蝶夫人》**：蝶衣与军官之间的爱情充满了矛盾和悲剧性，反映了情感的复杂性。

尽管两部作品在表现形式和具体情节上有很大不同，但它们都通过跨文化的背景和悲剧性的女性形象，探讨了人类情感的复杂性和文化差异带来的冲突。

杜拉斯的小说《恒河女》（《拉霍尔之旅》）是一部复杂而深沉的作品，情节围绕几个主要人物展开，通过不同的叙述视角探讨他们的情感和内心世界。主要剧情如下：

### 主要人物
1. **法国副领事（拉霍尔）**：他因精神问题和行为失控被调到加尔各答，表现出极端的孤独和边缘化。

2. **安娜-玛丽·斯特雷特兰德**：她是法国副领事的情人，一个神秘而富有魅力的女性。

3. **查尔斯-罗斯**：一个与安娜-玛丽有过复杂关系的英国外交官。

4. **恒河女**：一个被社会抛弃的女子，她的流浪故事贯穿小说，象征着绝望与孤独。

### 情节概述
1. **法国副领事的失控**：法国副领事因情绪失控和无法适应新环境而被调到加尔各答，他的行为越来越古怪，逐渐被周围人孤立。

2. **安娜-玛丽的纠葛**：安娜-玛丽是副领事的情人，她对副领事的感情复杂，既有同情又有无奈，同时她与查尔斯-罗斯的关系也不断揭示。

3. **恒河女的流浪**：恒河女的故事贯穿整部小说，她从恒河上游一路流浪到加尔各答，经历了极端的贫困和孤独。她的形象象征着被社会抛弃的女性命运。

4. **多重视角**：小说通过不同人物的视角，展示了他们对恒河女的不同理解和反应，揭示了更深层次的文化冲突和个人内心的挣扎。


### 主题和象征
- **孤独与边缘化**：副领事和恒河女的故事都表现出极端的孤独和与社会的疏离。
- **文化冲突**：小说中的人物处在不同文化之间，表现出对异国文化的陌生和冲突。
- **女性命运**：恒河女的形象象征着被社会忽视和压迫的女性，她的故事揭示了女性在社会中的困境和无助。

《恒河女》通过复杂的叙述结构和多重视角，探讨了孤独、痛苦和人类情感的复杂性，描绘了一个充满矛盾和张力的印度社会背景下的故事。
