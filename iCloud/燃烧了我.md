#ICA 影片改编自Cesare Pavese于1947年出版的诗集《Dialoghi con Leucò》中的一章“Sea Foam”，讲述了古希腊女诗人Sappho和仙女Britomartis在海边相遇并且产生的一系列对话。

角色开场重命名那部分非常有意思，两位角色通过现代方式达成了Sappho和Britomartis的重塑，再利用诗把他们交织在一起实在是浪漫，而片中的一些重复就如同那逃不开的宿命，这结尾对照了两位女性的宿命也呼应了Pavese的命运。

视听上，碎片化的视觉呈现效果是极其惊人的，蒙太奇的使用非常棒，尤其是大量空镜的呈现，还有利用35mm胶片与16mm胶片两种不同的胶片质感呈现出那不同的时代对话，空镜海浪冲刷海滩的泡沫，阳光下的绿树，那近乎90度仰拍到旋转的悬崖都能帮我带入到那诗句和神话中。片中重复的场景我觉得可以理解为诗句的视觉符号化，那一句句诗变成了那转不开的门锁，喷着水的喷泉，地下的蜜蜂涂鸦和一波波冲击的海浪。当然到结尾处这些画面也都慢慢的被补全。片中最美的肯定是在诗上一笔笔勾勒出的一副美丽诗中场景。电影中大量自然的声音与诗句独白结合，呈现出的张弛有度的感觉很好，尤其是海浪打在岩石上仿佛一种咆哮，诗句中蕴含的力量则比这股海浪还有强大。

影片像是一部散文诗电影，最厉害的不是整体呈现的先锋性，而是以整部电影呈现出视觉的诗意和与观众互动的形式呈现的诗。最震撼的一幕是诗集中的句子一遍遍的重复后被符号化刻入观众记忆中后的无声处理，让画面的重组让句子也重组后在观众的内心中响起实在是太牛了。

萨福（Sappho）是古希腊著名的女性诗人，约公元前630年出生于莱斯博斯岛（Lesbos）。她的作品大多是抒情诗，内容常涉及爱情、友谊和自然，被认为具有深刻的情感表达和精致的语言艺术。虽然她的诗作大部分只以片段的形式保存下来，但她的影响力仍然十分深远。

萨福在古代被誉为“第十位缪斯”，与阿尔克曼（Alcman）、阿尔凯奥斯（Alcaeus）、阿那克里翁（Anacreon）等抒情诗人齐名。她的诗歌风格独具特色，通常以个人视角表达情感，因此被视为个人抒情诗的开创者之一。

萨福的历史背景是古希腊的阿卡伊亚（Aeolian）文化，她在莱斯博斯岛上建立了一所学校，专门教授年轻女性文学和音乐。虽然关于她的生平和作品的具体细节较少，但她对古希腊文学的贡献是不可忽视的，其诗歌影响了后世许多作家和诗人。

萨福的诗歌多采用“萨福体”——一种由三行11个音节和一行5个音节组成的四行诗节。她的作品在内容上丰富多样，既有描写爱情的甜美和苦涩，也有对自然美景的细腻描绘。

总的来说，萨福在古希腊文学史上占有重要地位，她的作品不仅反映了古代希腊的文化和社会，也展现了她个人独特的艺术才华。

萨福的诗作大部分以残片的形式保存下来，以下是其中几首较为著名的作品：

1. **《阿佛罗狄忒颂》（Ode to Aphrodite）**：
   这是萨福最完整的一首诗，呼唤爱情女神阿佛罗狄忒帮助她赢得心上人的爱情。诗中充满了恳切的祈求和感人的情感表达。

   **原文（片段）**：
   ```
   彩虹般的阿佛罗狄忒，永恒的荣耀女神，
   ...
   来吧，来到我身边，解救我，
   实现我所有的愿望；成为我的伙伴，在这场斗争中。
   ```

2. **《月亮和群星》（Moon and Stars）**：
   这首诗描绘了夜晚的美丽景象，月亮和群星照耀大地。

   **原文（片段）**：
   ```
   月亮落下去了，
   群星也落下去了，
   凌晨的时间过去了，
   ...
   我孤独地躺着。
   ```

3. **《她似乎如神一般》（He Seems to Be a God）**：
   这首诗描述了一位男子在她心爱的女子面前的优雅与美丽，引发了她深深的情感共鸣。

   **原文（片段）**：
   ```
   他似乎如神一般，那人，
   当他坐在你身旁，
   静静地聆听你柔声的谈话，
   ...
   我的舌头断裂。
   ```

4. **《爱情是一种痛苦》（Love is a Torment）**：
   这首诗表达了爱情带来的甜美和痛苦，表现了萨福对爱情的复杂情感。

   **原文（片段）**：
   ```
   爱情是一种痛苦，
   突然袭来，
   就像狂风中的橄榄枝，
   ...
   被风吹散。
   ```

虽然萨福的许多诗作只以片段形式保存下来，但她的抒情诗以其深刻的情感表达和优美的语言著称，影响了后世无数的诗人和作家。



有的，萨福有一首诗涉及到蜂蜜和蜜蜂，尽管只是一个片段，但它同样充满了诗意。这个片段通常被称为“关于蜜蜂的诗”，如下：

**原文（片段）**：
```
她说，“蜜蜂是爱人的使者，
轻盈地从花丛中采集甜蜜，
在爱的花园里，
蜂蜜流淌，甜美而浓郁。”
```

这个片段反映了萨福对自然和爱情的细腻观察，将蜜蜂和蜂蜜的意象与爱情联系在一起，表现出她作品中常见的抒情风格和自然意象的运用。萨福擅长通过自然景物表达深刻的情感，使她的诗歌在简洁中蕴含丰富的内涵。


萨福在其诗歌中提到了许多与她关系密切的人物，这些人包括她的朋友、学生和家庭成员。以下是一些在她的作品中频繁出现的人物：

1. **阿提斯（Atthis）**：
   - 阿提斯是萨福的朋友或学生，萨福在许多诗歌中表达了对她深厚的感情。有些诗歌中表现出她们之间的亲密关系和分离的痛苦。

2. **安德罗美达（Andromeda）**：
   - 另一位萨福诗歌中的重要人物，可能是她的学生之一。萨福对她同样有着深厚的情感。

3. **克莉斯（Cles）**：
   - 可能是萨福的另一位学生或亲密朋友，萨福在诗中对她的美丽和魅力进行了歌颂。

4. **家庭成员**：
   - 萨福的兄弟拉里库斯（Larichus）和卡拉克斯（Charaxus）也出现在她的诗歌中。特别是她对卡拉克斯的关注和关心在诗中有所体现。

5. **阿尔卡埃乌斯（Alcaeus）**：
   - 古希腊另一位著名的抒情诗人，也来自莱斯博斯岛。虽然没有确凿的证据证明他们之间有密切关系，但一些文献和传说暗示他们可能相互认识并有一定的交流。

6. **埃里娜（Erinna）**：
   - 据说是萨福的学生，后来成为著名的诗人。尽管有关埃里娜的信息较少，但她的名字在与萨福相关的文献中出现过。

萨福的诗歌充满了对这些人物的深情描述，无论是她的朋友、学生还是家庭成员，都反映出她对亲密关系的重视和对情感表达的敏感。通过她的作品，我们可以看到她与这些人物之间的紧密联系和丰富的情感交流。


萨福的诗作主要以片段的形式保存至今，因为大多数她的作品在历史的长河中遗失了。不过，以下是萨福作品的一些主要集合和重要片段，它们被汇编成书籍或文集：

1. **《萨福的诗歌全集》（Complete Poems of Sappho）**：
   - 这类书籍通常收集了所有已知的萨福诗歌片段，并根据现存的文献和考古发现进行编排和注释。

2. **《希腊抒情诗》（Greek Lyric Poetry）**：
   - 许多关于希腊抒情诗的合集都会包含萨福的作品。这些合集通常会收录萨福以及其他同时代抒情诗人的作品。

3. **《萨福片段》（Sappho: A New Translation of the Complete Works）**：
   - 这类书籍提供了对萨福现存诗歌片段的最新翻译和解读，帮助读者更好地理解她的作品。

4. **《萨福诗选》（Sappho: Poems and Fragments）**：
   - 这类书籍选编了萨福最重要和最有代表性的诗歌片段，通常伴有详细的注释和分析。

萨福的诗歌主要通过古代文献和纸莎草文稿流传下来，现存的片段包括：

- **《阿佛罗狄忒颂》（Ode to Aphrodite）**：
  - 这是萨福最完整的一首诗，赞美爱情女神阿佛罗狄忒。

- **《她似乎如神一般》（He Seems to Be a God）**：
  - 描述了一位男子在她心爱的女子面前的优雅与美丽。

- **《月亮和群星》（Moon and Stars）**：
  - 描绘了夜晚的美丽景象。

- **《爱情是一种痛苦》（Love is a Torment）**：
  - 表达了爱情带来的甜美和痛苦。

这些作品通常通过研究和翻译被现代学者整理和出版，使得现代读者能够接触和欣赏萨福的诗歌艺术。
