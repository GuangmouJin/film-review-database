# 惺惺相惜


以下是对2024 年克洛德·巴拉斯（Claude Barras）导演的动画电影 **《Sauvages》（英名 *Savages*）**的全面分析：


	•	**生态正义与文化救赎**：森林代表文化与自然的生命之源，Oshi 象征被掠夺的生态与无辜，经济公司象征资本主义的殖民掠夺   。

	•	**身份与代际联系**：Kéria 与母系文化隔离感，使她在 Oshi 与 Selaï 中找到归属与自我认同；

	•	**矛盾的现代性**：影片呈现 Penan 社会虽不完全现代化，却具有主体性与科技适应能力（如长辈使用手机）、反衬资本机器的非人性 。


#ICA 周六五场连看第一场，影片传递的信息很明确，但是这种对立解决上仍然是含糊其辞，甚至感觉对于处理上推动的方向感觉不是很好，大公司是政府的意志，但只展现个体，而不是体系。看了个开始以为是《唯一的家园》+《勿忘迈因》，结果看到结尾发现是动画梦幻版的《阿凡达》，残酷不足，2022年还要文明与野蛮的问题感觉也有些太“落后”了，而且环保问题的教育上似乎也感觉怪怪的，代际关系的呈现上也不咋好，还有就是这个结构感觉也是英雄之旅，实在是不喜欢。黏土定格动画制作，整体的质感还是不错的，感觉特效的部分也不会比较突兀，环境音的设计做的很好，尤其是雨林与现代城市的区别呈现，片中爷爷的铃声还是有些搞笑的。


主题深度不够

#ICA 周六五场连看第一场，影片太格式化了，传递的信息很明确，但是这种对立解决上仍然是含糊其辞，甚至感觉对于处理上推动的方向感觉不是很好，大公司是政府的意志，但是并没有得到强调，只展现了“个体”，而不是法律体系。看了个开始以为是《唯一的家园》+《勿忘迈因》，结果看到结尾发现是动画梦幻版的《阿凡达》，残酷不足，2022年还要文明与野蛮的问题感觉也有些太“落后”了，而且环保问题的教育上似乎也感觉怪怪的，后来也想了想，感觉就是矛盾的现代性的问题，而且代际关系的呈现上也感觉很奇怪，父亲无法参与的部分也只能等待母亲的幽灵，这种等待是不是太死板了，还有就是这个结构感觉也是英雄之旅，实在是不喜欢。

视听上，黏土定格动画制作，整体的质感还是不错的，感觉特效的部分也不会比较突兀，环境音的设计做的很好，尤其是雨林与现代城市的区别呈现，片中爷爷的铃声还是有些搞笑的。

	•	**当代生态危机语境**：棕榈油种植破坏环境、侵犯原住民权利是广泛全球议题；
	•	**原住民发声与认同**：影片参与制作的 Penan 社区成员，使影片具高度文化尊重与自治表达；
	•	**全球儿童动画平台化趋势**：延续巴拉斯“儿童并不代表无知”的创作理念，将严肃议题以童视角呈现，兼顾成年人观看深度。
