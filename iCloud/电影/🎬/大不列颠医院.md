#BFI Mick Travis三部曲的最后一部，对照前两部看的话同样的讽刺，批判及荒诞，内容上算是结合了第一部《如果》的学校密闭环境和阶级视角和第二部《幸运儿》的社会全面性。从影片刚开始能明显感觉出影片对“左右”的输出，类似于效率低下，官僚主义，媒体失真，特权阶级，公共服务和资源分配不均（当然他也没有放过自己，嗨了的同事叫喊着要拍电影摔到了人群中）等，其实这部分能明显看出是撒切尔夫人时期的弊端展现，工业衰退，劳资冲突，抗抑油行，外交干预都在一家小小的医院体现了出来。但当影片中的几条线到最后汇聚的疯狂医生的作品时，我认为电影所讨论的议题其实是从社会上升到人类的层面的跨步的，对于社会的讽刺少了，但有更多的展现的是人类的狂妄及疯狂。

角色上，男主这次的作用其实是格外小的，更多的是作为一个配角，但是影片的角色模式跟前两部作品是一样的，每个角色都有所指代，以现在的情况看，这种对于现实割裂的讽刺是存在极强的预言性的。影片中的主角可以看作是那位疯狂医生，“输出”能力极强，他的肢体语言和表情都透露出了一个极端激进分子的所有特质（好喜欢菜刀和用牙咬断线的“专业性”展示）

风格上，整体的表达还是林赛自己的政治性，片中还有柯南伯格式的身体恐怖，当然英式幽默也占比极高，几个角色是作为专注认真的奇观出现。

视听上，均中规中矩，光影和色彩做的很好，当然还有仰拍的权力展现很明显。配乐则是带有很强的隐喻，手拉手唱歌那幕笑死。

唯一不太满意的可能是结尾太弱，可以理解表达，但在前面一切如此炸裂的情况下这个收尾还是单薄了些。

“左派”与“右派”在政治上的主要诉求确实可以大致归纳为“平等”和“自由”，但这两者的弊端各有所异。以下是对左派追求平等和右派追求自由的弊端的分析：

### 左派追求“平等”的弊端：

1. **经济效率低下**：
   - 在追求经济平等的过程中，左派政策可能会实施高税收和广泛的再分配措施。这些政策可能会抑制个人和企业的创新和投资，导致经济增长缓慢。

2. **资源分配问题**：
   - 平等政策常常需要政府在资源分配上起主导作用，但政府可能无法高效分配资源，导致浪费和腐败。

3. **激励机制减弱**：
   - 过度强调平等可能会削弱个人的动力，因为努力工作和创新所获得的回报会被平均分配。这可能会导致整体生产力下降。

4. **官僚主义和政府干预过多**：
   - 为了实现平等目标，左派政府可能会实施更多的监管和控制，这可能导致官僚主义和政府对个人生活和经济活动的过度干预。

### 右派追求“自由”的弊端：

1. **社会不平等加剧**：
   - 右派政策通常强调市场自由和个人责任，这可能会导致财富和机会的分配不均，富者愈富，贫者愈贫，社会不平等加剧。

2. **社会安全网不足**：
   - 在减少政府干预的同时，右派政策可能会削减社会福利和公共服务，导致弱势群体缺乏基本的保障和支持。

3. **市场失灵**：
   - 完全自由的市场有时会出现市场失灵，例如垄断、外部性和信息不对称等问题，这些问题需要政府干预来纠正。

4. **公共物品提供不足**：
   - 右派政策可能会减少政府在公共物品（如基础设施、教育和公共安全）上的投入，导致这些领域的发展滞后。

### 综合来看：

- **左派弊端**主要集中在可能导致经济效率降低、资源分配不当和激励机制减弱等方面。
- **右派弊端**则主要在于可能加剧社会不平等、公共服务不足和市场失灵等问题。

平衡这两种取向的优缺点，寻找适合具体社会背景和经济状况的政策，是各国政府在制定公共政策时面临的主要挑战。不同国家和社会需要根据自身的历史、文化和经济发展阶段来权衡和选择。



1982年的电影《大不列颠医院》（Britannia Hospital）是一部英国黑色喜剧电影，由林赛·安德森（Lindsay Anderson）执导，是“米克·特拉维斯三部曲”（Mick Travis Trilogy）中的最后一部，前两部分别是《幸运儿》（If....，1968）和《哦，幸运儿！》（O Lucky Man!，1973）。这三部电影都由马尔科姆·麦克道尔（Malcolm McDowell）主演。

以下是对《大不列颠医院》的评价和简介：

1. **剧情简介**：
   - 影片聚焦于一家象征性的大不列颠医院，在英国王室某成员的视察日，这家医院正面临种种混乱和问题。医院内部的医护人员在准备应对这次重要访问的同时，处理着来自病人、抗议者和管理层的多重压力。

2. **黑色幽默与讽刺**：
   - 电影延续了林赛·安德森一贯的黑色幽默和讽刺风格，通过夸张和荒诞的手法，揭露了英国社会和医疗体系中的各种弊病和矛盾。影片中，医院的混乱状况被比作整个国家的问题缩影，充满了对社会制度和官僚主义的辛辣批评。

3. **角色与表演**：
   - 马尔科姆·麦克道尔再次饰演米克·特拉维斯，这次他作为一名记者进入医院，揭示其内部的腐败和混乱。影片中还有许多其他令人难忘的角色，他们各自代表了社会中的不同群体和利益。

4. **社会与政治背景**：
   - 影片反映了20世纪70年代末和80年代初英国社会的动荡和不满，特别是对公共服务、医疗体系和政府政策的批评。它不仅是对医院管理的讽刺，更是对整个英国社会的一次深刻反思。

5. **导演手法**：
   - 林赛·安德森通过交织现实与夸张、讽刺与严肃的手法，使得电影在风格上独具一格。电影中的场景设计和情节安排充满了象征意义，使得观众在幽默中反思现实。

6. **电影的影响与评价**：
   - 尽管《大不列颠医院》在商业上并未取得巨大成功，但它作为一部具有深刻社会批判意义的电影，受到了影评界的关注。影片被认为是安德森对社会问题的大胆揭示和艺术探索。

总体来说，《大不列颠医院》是一部充满黑色幽默和社会批判的电影。通过对一个象征性医院的描绘，揭示了英国社会中的种种问题，是一部值得观看和思考的经典之作。
