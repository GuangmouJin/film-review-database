# 三只猴子


#ICA 现实主义作品，那灵动的一刹有一种阿彼察邦同样的感觉，感觉都很大程度的来源于阿巴斯的风格。父亲的顶罪就像一记响雷劈在了这关系本就极不稳定的家庭中，🙉🙊🙈的状态非常贴近于东亚的家庭环境（其实家庭这部分总让我想到伯格曼的电影），而顶罪的轮回象征着那不可逾越的阶级也预示着道德困境的传递。

视觉上，大量的固定镜头和摇镜头对压抑环境的呈现是极强的，尤其是室内360度环绕那场戏，逼仄的环境，压抑的情绪集中到一起（其实外景能注意到他们住的房子从外面看也只有一条），长镜头还很大程度的帮助了叙事。景别上，中近景其实蛮少的，大多是大远景，全景和特写的使用，呈现出的效果是非常棒的，全景的压抑，大远景的疏离和特写的压抑情绪交替呈现。光影和构图上，主角一家生活的房间非常有代表性，固定镜头利用门框的构图把角色间完全隔离开，而大量角色的背光则是利用后景的蓝天阳光把整个前景的环境都变得更加压抑，当然还有那整体发黄的配色，影片中也是少有几场出现亮色的场景，对于释放情绪有一定作用。几场升格用于表达那过去的裂痕对于家庭中每个角色的影响。

听觉上，整体还是喜欢，现场收音的效果极佳，尤其是那细微的环境音处理，海浪声风声等自然音效效果非常好。还很喜欢去除环境音的超现实处理和远景的现场收音效果。不太喜欢的部分是几场雷声有些过于刻意，再者就是有几处后配音的嘴形不太对。

《三只猴子》（Üç Maymun）是土耳其导演努里·比尔盖·锡兰（Nuri Bilge Ceylan）2008年执导的一部电影，受到了影评人的广泛关注和评价。以下是影评人对《三只猴子》的主要评价：

1. **视觉美学**：影评人普遍赞赏锡兰在《三只猴子》中展现的卓越视觉风格。电影的摄影、构图和色彩处理被认为非常出色，特别是电影中利用自然光线和阴影的手法，增强了影片的视觉冲击力和情感深度。

2. **叙事手法**：影评人对锡兰的叙事方式表示赞赏，尤其是其通过细腻的情感描写和人物刻画，深入探讨了人类的复杂性和脆弱性。电影通过一个家庭在面对危机时的反应，展现了人性中的弱点和挣扎。

3. **演员表现**：影评人高度评价影片中演员的表现。尤基·阿维纳（Yavuz Bingöl）、哈蒂丝·阿斯兰（Hatice Aslan）和里法特·桑贾尔（Rıza Kocaoğlu）等演员的表演被认为是自然且真实的，成功地传达了角色的内心冲突和情感波动。

4. **主题深度**：影片探讨了沉重的社会和道德问题，如责任、忠诚、罪恶和救赎。影评人认为，锡兰通过一个家庭的故事，揭示了更广泛的社会现实和人性真相，给观众带来了深刻的反思。

5. **国际认可**：《三只猴子》在2008年戛纳电影节上获得了最佳导演奖，进一步肯定了其在国际影坛上的重要地位。影评人普遍认为，这部电影不仅展示了锡兰的导演才能，也提升了土耳其电影在国际上的影响力。

总体而言，《三只猴子》被认为是一部视觉上精美、情感上深刻的电影，通过细腻的叙事和强烈的情感表达，深刻探讨了人性和社会问题，在影评界获得了高度评价。
