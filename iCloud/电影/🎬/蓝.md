# 蓝


#CurzonMayfair 还是蹲守到了这部电影，实在开心，“Oh, blue come in” 完全静态的蓝色充满屏幕，仿佛是一个超级长镜头，我们睁着眼睛所看的及时他所见的，“thinking blind, becoming blind”。影片内则完全地利用声音进行表达，诗意的台词和精致的配音实在太吸引人了（就是需要极其专注的听台词有些累人）。环境音，音效结合台词似乎可以构建出了一幅幅美丽的画面（其实看过《至爱梵高》之后，这蓝色似乎也有了油画的质感），远处服务员翻动刀叉的声音，关门声，汽车声，咒骂声和爆炸声，影片以萨拉热窝围城战开始讲述。轻声的低语和磬钵金刚铃频繁出现在影片中，似乎也展现出了即将接近的死亡，蒂尔达的声音特殊处理，空灵且带有神圣的感觉。随后出现的各式各样的环境音和音效把我们从一个场景拉到另一个场景海浪声的海边，石子声的小径，电子乐的柏林，印象最深的是由钟表声组成的“时间”（太过震撼）和讽刺歌词的合唱（合音效果拔群），To all thr true lovers。
