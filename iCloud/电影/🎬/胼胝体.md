# 胼胝体

# 
《**Corpus Callosum》 (2002)**

#ICA 看电影的时候有些走神，感觉可能因为需世上没有一个比较强的逻辑性，影片除了电影与其背后，更是加入了数字技术的思考。内容上依然是荒诞，角色充当了类似神经元的作用，影像风格不断变形和切换仿佛左脑与右脑的冲突与融合，感觉我们是随着角色在逻辑与直觉，现实与幻象之间游走。影片的镜头主要呈现了两个场景，一个办公室，一个是家庭，一个采用横移推轨，一个采用固定镜头，感觉二者不光视觉呈现不同，也利用着特有的元素呈现出不同的讽刺，而角色的游走则更佳呼应了影片的片名。影片也依然延续导演其他作品强调的“在场”，强调“现实”，戏中戏的设定让人一阵失神，家庭部分的镜子是最稳定的，而摄影机在其中，办公室中的数字技术侵入呈现出的效果则更像强调了影像的人为性，二者结合像是胼胝体失调的结果。不得不感慨导演的预见性。


2. 现实与幻象的边界
	•	斯诺曾谈到 《Corpus Callosum》 是他对现实与虚拟影像之间界限的研究，尤其是在 数字时代，影像不再是“现实的记录”，而是可以被无限操控、变形的虚拟对象。
	•	电影中的办公室、家庭场景，表面上是普通的日常空间，但它们总是在微妙地变形、拉伸、扭曲，人物的身体也不断被拉长、压缩、变形或消失，形成一种不稳定的现实。
	•	这些数字变形效果不是为了制造“奇幻”感，而是为了揭示影像的人工本质——即我们所看到的“现实”只是影像制造的幻象，而影像本身也并不稳定。

	“电影是一个系统，它塑造了我们对现实的看法，而数字技术进一步让现实变得可塑和不确定。” —— 迈克尔·斯诺

3. 电影的观看体验：感知的操控
	•	斯诺强调，他希望观众在观看《Corpus Callosum》时，体验到一种持续的知觉变化——画面不断变化，但没有明确的叙事逻辑，观众的注意力也因此被不断挑战。
	•	影片的节奏、视觉效果、声音与现实的对比，使得观众的体验既沉浸又疏离，形成一种类似**“知觉游戏”**的状态。
	•	斯诺认为，这种体验类似于“胼胝体”在左右脑之间传递信息时的动态过程，因此，这部电影不仅是关于影像的变化，也是关于观众如何“感知”影像的变化。

4. 斯诺对“超逼真影像”的批判
斯诺在《Corpus Callosum》中，也回应了他对“超逼真影像”的看法：
	•	他对现代影像技术（如3D、IMAX、全息影像）并不盲目推崇，反而带有批判性。他认为，这些技术追求越来越“真实”的影像，但实际上，它们只是更复杂的幻象，让人更难区分真实与虚拟的边界。
	•	在对谈中，他提到，如果这种影像技术继续发展，未来可能会出现完全逼真的虚拟世界，甚至让人“绕过感官系统，直接把影像植入大脑”——这与《Corpus Callosum》中影像的变形和操控可以看作是一种隐喻。

5. 斯诺对这部电影的总结

在访谈中，斯诺曾总结 《Corpus Callosum》 是他对电影本质的一次深度实验：

	“这是一部关于‘变形’的电影，关于图像如何变化，以及我们如何适应这种变化。”

	•	他希望通过影像的不断扭曲和变化，让观众意识到影像的人工性，而不是沉浸在‘幻象’之中。
	•	他认为，电影不是单纯的**“讲述故事”，而是可以让观众主动探索、思考“我们如何观看”**。
	•	这是一种关于感知的体验，而不是一部传统的叙事电影。



融合了斯诺电影作品中一贯的主题和特征，*Corpus Callosum* 探索了模拟与数字、幻象与现实之间的界限。摩天大楼的上层办公室里，员工（大多）坐在工位上；家中的一家人围坐在电视机前；然而一切都不像表面看上去那么简单。

影片以连接大脑左右半球的胼胝体（*Corpus Callosum*）命名。斯诺的最后一部长篇作品——在影评人乔纳森·罗森鲍姆看来，是他“最伟大的电影”——以迷幻的方式勾勒出 21 世纪初的生存体验。


在 《Corpus Callosum》 中，办公室场景和家庭场景确实可以分别象征逻辑和直觉，但影片并没有严格地将二者割裂，而是通过影像的变形与错位，使它们相互渗透、不断模糊。这种处理方式，与胼胝体（Corpus Callosum）作为连接左右脑的桥梁的概念相呼应，使得影片本身也成为了一种“左右脑交互体验”。

1. 办公室场景：逻辑、秩序、僵化

象征“左脑” —— 逻辑、分析、规则、计算
	•	影片中，办公室的员工们大多坐在工位上，表现出一种受控、机械化的状态，对应现代社会中被规训的工作环境。
	•	视觉风格上，这些场景强调结构化的空间（办公桌、电脑、整齐摆放的物品），给人一种受逻辑支配的秩序感。
	•	然而，这种秩序并不稳定：人物的身体开始拉伸、扭曲、变形，办公室空间也变得流动，打破了它原有的稳定结构。
	•	斯诺通过这种影像变形暗示：即使是最具逻辑性、最有秩序的世界，也无法完全控制现实，它仍然受“幻象”影响，并可能崩解。

2. 家庭场景：直觉、情感、混沌

象征“右脑” —— 直觉、感受、情绪、梦幻
	•	家庭成员围坐在电视机前，似乎是在进行一种被动的、沉浸式的观看，这与办公室的“工作模式”形成对比。
	•	这里的视觉风格更柔和，家庭空间相比办公室更“温暖”，但同样是不稳定的：家居环境会突然变化，人物的身体也经历变形和消失，现实与幻象在这里更难以分辨。
	•	这可能象征了“直觉”和“情感”虽然不像逻辑那样受结构控制，但同样无法避免被幻象影响，甚至可能成为影像操控的牺牲品（如电视画面成为支配现实的存在）。

3. 影像变形：逻辑与直觉的交汇

如果说办公室象征左脑，家庭象征右脑，那么整个电影的“变形”过程就像是胼胝体在连接和协调两者：
	•	影片中，办公室场景和家庭场景都经历了失序、变形、扭曲、流动，这种方式恰恰揭示了逻辑和直觉并不是完全对立的，而是在相互影响、相互渗透。
	•	例如，办公室的变形不仅仅是一种秩序的破坏，同时也带有某种幽默感（类似斯诺在《Sshtoorrty》中使用的闹剧元素），这让本应是“理性”的空间变得荒诞，左脑逻辑的“严肃性”被削弱。
	•	同样，家庭场景虽然象征直觉和情感，但电视机的存在提醒观众：这种直觉体验本身也是受影像控制的，即直觉并不总是“自由的”，而可能被技术和媒介所塑造。

4. “胼胝体”作为整体体验

斯诺希望这部电影不仅仅是一个关于“逻辑 vs 直觉”的二元对立，而是让观众在观看过程中体验到这两种模式如何相互交融，并最终模糊彼此的界限：
	•	逻辑（办公室）试图保持秩序，但秩序最终被影像变形所摧毁，成为某种荒谬的混沌；
	•	直觉（家庭）被设定为自由和感性的空间，但家庭成员面对电视机的画面时，他们的感知同样受到影像的控制；
	•	电影本身就像是胼胝体的功能，它连接、协调、变形、扭曲、流动，让观众在左脑和右脑的不同模式之间来回切换。

5. 斯诺的观点：超越二元对立

斯诺在访谈中曾提到，他对逻辑和直觉的关系并不做绝对的划分，而是想让观众去感受它们之间的流动性：

	“现实和幻象的界限已经变得不清晰，我们的知觉方式也在不断被影像重塑。”
“这不仅仅是逻辑和直觉的问题，而是关于我们如何‘知觉’影像，以及影像如何影响我们的思维方式。”

因此，《Corpus Callosum》并不是在强调办公室 = 逻辑、家庭 = 直觉的简单关系，而是：
	1.	逻辑和直觉都受到影像操控，都不是“绝对可靠”的知觉模式。
	2.	影像的变形过程展现了知觉的动态性，使观众无法稳定在某一个观看方式上。
	3.	电影本身成为了“胼胝体”的体验，连接两种不同的知觉模式，让观众在观看过程中不断转换自己的理解方式。

结论
	•	办公室代表逻辑（秩序、分析），家庭代表直觉（感性、沉浸），但两者并非截然对立，而是在影片中不断相互渗透、交融。
	•	影像的变形让逻辑与直觉的界限变得模糊，强调了我们的知觉并不是单一的，而是受影像塑造和影响的动态过程。
	•	斯诺用这部电影模拟了胼胝体的功能——它连接、协调、改变我们如何感知现实，使观众在观看过程中也经历这种知觉的错位与流动。

因此，观看 《Corpus Callosum》 既不是单纯的理性分析，也不是完全的感性体验，而是一种不断在逻辑与直觉之间跳跃的“胼胝体式”观影体验。

为了继续这个话题进行推测：如果“进步”持续发展，一个科学精英阶层将继续负责研究、发明和娱乐（毕竟，人类的登月之旅最终成了一场电视节目）。而剩下的大众——即那些如今每天花 23 小时盯着电视屏幕的人们的后代——或许会通过全然逼真的影像“参与”其中，而这些影像甚至可能绕过感官系统，直接插入大脑之中。



是的，迈克尔·斯诺曾对 《Corpus Callosum》 (2002) 提供了一些解释，并表达了他的创作理念。这部电影延续了他一贯对感知、幻象、现实与影像本质的探索，同时结合了数字技术对影像的操控，使得观众的观看体验变得更加流动和不稳定。以下是他的一些核心观点：

1. 关于电影标题 “Corpus Callosum”（胼胝体）

斯诺在访谈中提到，他选择以 “胼胝体” 命名，是因为这是一部关于连接、传递、变化和割裂的电影，而胼胝体正是大脑中连接左、右半球的神经纤维结构。

	•	影片的影像风格不断变形、切换，仿佛左脑与右脑的冲突与融合，观众在逻辑与直觉、现实与幻象之间游走。
	•	视觉和听觉的错位 让观众不断调整自身的感知方式，正如大脑在处理矛盾信息时的反应。
	•	他还指出，胼胝体的功能如果被切断（如“裂脑实验”），会导致人的知觉与认知发生奇特的分裂现象，这种分裂感也正是影片想要制造的体验。

2. 现实与幻象的边界

	•	电影中的办公室、家庭场景，表面上是普通的日常空间，但它们总是在微妙地变形、拉伸、扭曲，人物的身体也不断被拉长、压缩、变形或消失，形成一种不稳定的现实。
	•	这些数字变形效果不是为了制造“奇幻”感，而是为了揭示影像的人工本质——即我们所看到的“现实”只是影像制造的幻象，而影像本身也并不稳定。

	“电影是一个系统，它塑造了我们对现实的看法，而数字技术进一步让现实变得可塑和不确定。” —— 迈克尔·斯诺

3. 电影的观看体验：感知的操控
	•	斯诺强调，他希望观众在观看《Corpus Callosum》时，体验到一种持续的知觉变化——画面不断变化，但没有明确的叙事逻辑，观众的注意力也因此被不断挑战。
	•	影片的节奏、视觉效果、声音与现实的对比，使得观众的体验既沉浸又疏离，形成一种类似**“知觉游戏”**的状态。
	•	斯诺认为，这种体验类似于“胼胝体”在左右脑之间传递信息时的动态过程，因此，这部电影不仅是关于影像的变化，也是关于观众如何“感知”影像的变化。

4. 斯诺对“超逼真影像”的批判

斯诺在《Corpus Callosum》中，也回应了他对“超逼真影像”的看法：
	•	他对现代影像技术（如3D、IMAX、全息影像）并不盲目推崇，反而带有批判性。他认为，这些技术追求越来越“真实”的影像，但实际上，它们只是更复杂的幻象，让人更难区分真实与虚拟的边界。

5. 斯诺对这部电影的总结

在访谈中，斯诺曾总结 《Corpus Callosum》 是他对电影本质的一次深度实验：

	“这是一部关于‘变形’的电影，关于图像如何变化，以及我们如何适应这种变化。”

	•	这是一种关于感知的体验，而不是一部传统的叙事电影。

结论

迈克尔·斯诺并没有为 《Corpus Callosum》 提供单一、固定的解释，而是希望观众在观看过程中自行思考和感受。但他确实提到：
	1.	电影的变形和扭曲对应了胼胝体在左右脑间传递信息的过程，也隐喻了现实与影像的关系。
	2.	影像的**“不稳定性”** 反映了我们对现实的认知如何被影像塑造和操控。
	3.	电影不仅是关于影像本身，更是关于观众如何观看影像。

最终，《Corpus Callosum》 是一部关于影像、感知和认知边界的实验电影，它挑战了我们习惯的观看方式，并探索了现实与虚拟之间日益模糊的界限。
