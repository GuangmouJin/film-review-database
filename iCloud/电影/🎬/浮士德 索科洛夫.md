# 浮士德 索科洛夫

浮士德 索科洛夫

#ICA 影片以歌德的文学作品作基础，电影视听与主题契合度非常高，空间，色彩和内心独白等共同构建了一部带有存在主义思考的公路电影，内核则带有着对于“上帝已死”后的迷茫与错愕，如同一个预言，一个疑问。风格上影片是带有一些极简主义风格的特点，大量的内心独白直接呈现角色，克制的情绪表达，局部特写的跳剪等。片中不光对人类对权力和欲望无尽追求的批判，也揭示了欲望与邪恶。叙事结构算是一部公路片，由于没看过《浮士德》原著所以不清楚这种叙事形式是否跟原著相同，但这种“游”的形式让我想到了但丁的《神曲》中维吉尔游地狱的形式。

角色上，批判力度非常强，不光是对于浮士德和社会，甚至还有墨菲斯托这个恶魔，浮士德出售灵魂以换取的一切在影片中段可以理解为对于欲望的追求，以至于在结尾处呈现的“上帝已死”的言论在影片结尾处显得如此讽刺。影片的宗教色彩也非常强，感觉也不光是因为原著，索科洛夫本身的作品就有很强的宗教感，有其他本人是一名东正教徒。影片有一幕墨菲斯托亲吻圣像那一幕能听到影厅中非常明显表达不适的声音。墨菲斯托这个角色设计的和演绎的都非常好，这个形象应该是与原作的形象有很大不同的，影片中呈现出的形象是丑陋，冷酷和猥琐的，其中的贪婪特质肯定是最贴切的原著的，但如此的形象却利用剪辑的台词和行动展现出了别样的魅力，这也是让人我感觉美妙的原因，而且引导者的身份在结尾所完成的表达更是对于浮士德一个升华。

视觉上，最精彩的部分，首先是空间上，极致的空间扭曲，尤其是与墨菲斯托同行的大量场景，不光是广角镜头对于画面的拉扯，那种夸张的变形效果让人物和场景看上去既压迫又怪诞，甚至还有画面存在于另一空间呈现出的“非真实”感，还有空间的压抑呈现，片中不光利用大量的布景和镜头角度对于空间进行压缩，窄小和封闭的构图增加了视觉上的压迫感，甚至还让角色间贴合到一起造成一种带头欲望性质的压抑状态。然后是色彩的呈现于转变，冷色调的画面和压抑的场景布置将浮士德的心理状态和内心的渴望具象化，色彩暗淡、单调，进一步营造了压抑的情绪氛围。当然影片也是出现过一些暖色调，不过那种颜色呈现有些超过阈值的感觉，过爆的场景感觉主要还是包含讽刺。柔焦和失焦，电影中大量的柔焦和失焦效果呈现出一种画面轮廓不清晰的视觉效果，仿佛一切都笼罩在一种虚幻和梦境的氛围中，感觉这也是浮士德的主观迷惘的呈现。

听觉上，依然是很犹豫的地方，一方面是配乐很美妙，内心独白的呈现效果也很好，还有音效呈现出的恶魔低语都很喜欢，但另一方面就是后配音的部分不适感非常强。

映前Intro的嘉宾说他的作品受到了19世纪的一位画家Carl Spitzweg的启发，感觉从暖色的的部分看还是非常相似的，尤其是最震撼的那两幕特特写了镜头，但前后很多阴冷潮湿和冷色感觉还是有很大的不同的，感觉跟文艺复兴时期的很多作品相似，尤其是那兔子和猴子，让我想起了Albrecht Dürer。还注意到影片中出现白鹤，之前看《自己去看》中也出现过，在俄罗斯预示着神圣与美好。




在亚历山大·索科洛夫的电影《浮士德》中，批判的核心依然围绕着人类对权力、欲望、和存在意义的无尽追求，但他以独特的方式深化和扭曲了这些主题，将人性中的阴暗面和道德腐化推向极致。索科洛夫的电影语言和歌德的文学表现有所不同，他采用压抑、扭曲的镜头来探索人类精神的脆弱和扭曲。通过视觉的象征性和暗色调的氛围，他不仅是对人类追求的无底深渊进行了描绘，也提出了一种对人性堕落的深刻批判。

总体而言，索科洛夫的《浮士德》不仅是对人类欲望和追求无意义的批判，更是对人性本质的深刻挖掘。他的电影让观众直视欲望的黑暗面、权力的诱惑与人类存在的荒诞感，用一种几近冷酷的方式揭示出人类在追求中的迷失与堕落，将批判从个人扩展到对人类本质的哲学反思。

索科洛夫的《浮士德》给人的感觉犹如一场被梦魇侵蚀的意识之旅，电影没有直接地讲述灵魂交易的简单故事，而是用一种残酷而缓慢的方式，将浮士德对知识和欲望的极端追求揭开，直至观众被拖入他那内在腐蚀的深渊。这部影片不仅仅是歌德的改编，索科洛夫更以冷酷的哲学视角将浮士德化作一面镜子，让人看到欲望与人性暗影的荒谬。

在视觉上，影片将空间扭曲到极致，鱼眼和广角镜头拉扯着画面边缘的每一个元素，仿佛世界本身已变形腐坏。索科洛夫的用镜头语言仿佛在暗示，浮士德所在的每一片空间都已被他内心的无底欲望侵蚀，观众随着镜头的拉伸而不由自主地被拖进这场压抑的噩梦。过曝的光线犹如利刃般穿透每一处阴影，极具侵略性的光影似乎在揭示浮士德那极度渴求知识的欲望，明亮却刺眼、无所顾忌，最终把一切都推向自我毁灭的边缘。

声音上，索科洛夫近乎沉默的处理更让影片在压抑的空间中显得格外空旷。寂静的音效中隐隐传来如同低语般的回响，仿佛欲望在每一个角落都悄然蔓延。浮士德与魔鬼梅菲斯特之间的对话尤为冷峻，他们的言辞简单，却带有一种无法抗拒的压迫，仿佛在极静中酝酿出一场无声的崩溃。没有直接的情感引导，这种寂静让每一场对白更具重量，仿佛一切都在无尽的沉默中走向不可挽回的深渊。

在浮士德和梅菲斯特的关系中，索科洛夫对权力和欲望的批判带有极强的冷嘲。他通过浮士德逐渐加深的自我迷失，揭示人类对权力与知识的贪婪不仅是自我毁灭的开端，更是一场漫无止境的腐蚀过程。浮士德在欲望的驱使下不仅迷失自我，甚至最终化为一具空壳，他的存在仿佛只剩下欲望本身，一种不死不灭却永不满足的存在。

索科洛夫在这部作品中批判的核心并非对知识的追求，而是对追求本身荒谬性的揭示。他把人性之暗带到一个极端，通过压抑的画面和残酷的沉默使观众逐渐意识到，那些我们认为能够填补空虚的渴求，终究是将人推向无尽迷失的无底深渊。《浮士德》最终给予观众的不是答案，而是一个反复无解的质问：在无限的追求中，人类究竟在追求什么？这种问题没有答案，却让观众在索科洛夫构筑的暗影迷宫中不断回味。


浮士德的叙事确实在某种程度上与但丁《神曲》中维吉尔带领但丁游历地狱的形式相似，但两者的侧重点和表达方式有所不同。

在但丁的《神曲》中，叙事形式是明显的旅程结构：维吉尔作为导游带领但丁游历地狱、炼狱和天堂，逐步揭示罪与救赎的哲学意义。这种带有“指引”的旅程结构是《神曲》的核心，目的是通过层层深入的观察揭示人类灵魂的堕落和净化过程。

在《浮士德》中，叙事形式虽然没有完全采用《神曲》那样的旅程结构，但浮士德与魔鬼梅菲斯特的关系具有类似的引导性质。梅菲斯特作为浮士德的“引路人”，带领他经历各种欲望和诱惑，仿佛是对地狱和人性黑暗面的“游历”。这种“游历”并非空间上的层层递进，而是心理和道德上的深入。梅菲斯特引导浮士德从学术追求到欲望享乐，再到更高理想的追求，每个阶段都像是对人类欲望的逐步探索，就像《神曲》中对罪孽的层层揭示。

然而，与《神曲》不同的是，浮士德的旅程并没有明确的“导向”或“救赎”目标。浮士德并没有被带领去理解罪孽和救赎，而是深陷于欲望的诱惑之中，体验了内心的失衡和痛苦。他的旅程更像是内在世界的流浪，没有明确的终点和结构化的救赎安排，带有一种存在主义的荒诞感。

总的来说，浮士德的叙事形式虽然在角色关系上与但丁的《神曲》有相似之处，但本质上更注重心理的自我探索和道德的迷失，而不是通过引导来达成对罪与救赎的认知。

亚历山大·索科洛夫的电影《浮士德》是一部带有强烈个人风格的作品，以歌德的经典文学作品为基础，但赋予了其独特的俄式哲学深度。索科洛夫以极简主义和富有美学的黑暗镜头，将浮士德对知识和欲望的渴求表现得既极端又痛苦。影片中充满了扭曲的镜头角度和精心布置的场景，给予观众一种压迫感，同时也带有一丝超现实的梦幻色彩。

《浮士德》不仅仅是对灵魂交易的故事演绎，更是索科洛夫对人性复杂性的挖掘。他的镜头语言往往透过人物内心的自我挣扎和痛苦，去揭示人类存在的荒谬和痛苦。在视觉上，影片色调阴暗，画面质感沉重，给人一种幽闭恐惧的感觉，而音乐的处理则更显得沉默寡言，犹如在回响中消逝的低语，突显了人物的孤独和内心的绝望。

索科洛夫的《浮士德》不仅是在文学经典基础上的重构，也是对人类自身灵魂的质问。他探讨了人类对知识、权力和欲望的无止境追求，以及这份追求所带来的自我毁灭和道德崩溃。在浮士德最终选择放弃一切之际，观众既感到解脱又无法避免地陷入沉思，反思自我与存在的价值。

浮士德的故事源自于德国民间传说，并经由作家们不断演绎，最终成为经典的文学主题。最著名的版本由德国作家约翰·沃尔夫冈·冯·歌德（Johann Wolfgang von Goethe）创作，分为《浮士德》第一部和第二部。这一故事通过浮士德博士与魔鬼梅菲斯特的契约，揭示了人类对知识、欲望、幸福的永恒追求及其伴随的道德困境。

故事的核心人物浮士德是一位渴望知识的学者，年老体衰，经历了知识上的极限，感到空虚和无意义。为了满足自己对人生奥秘的好奇和对世俗欲望的追求，他召唤魔鬼梅菲斯特。梅菲斯特向浮士德承诺，能帮助他体验世界上所有的快乐与满足，但作为交换，浮士德必须在死后将灵魂交予魔鬼。浮士德接受了契约，开始了一段极富戏剧性的历程。

在歌德的《浮士德》第一部中，浮士德的灵魂挣扎和道德冲突通过他的爱情故事展现出来。浮士德遇见了美丽的少女玛格丽特，并与之坠入情网。然而，这段关系最终导致了玛格丽特的悲剧性毁灭，而浮士德在迷失中遭受了良心的拷问。

《浮士德》第二部则走向更为哲学和象征性的层面，浮士德的追求从个人欲望转向更广泛的理想，他试图通过创造和统治来实现“更大的善”。然而，这样的欲望最终也无法满足他对生命本质的探求，甚至将他引向毁灭性的结局。在这个过程中，浮士德的灵魂仍在挣扎，而他与梅菲斯特的斗争也变得更具象征意义。

浮士德的故事不仅探讨了个人追求的道德界限和人性中的阴暗面，更是对人类存在意义、灵魂救赎以及人生终极价值的深刻反思。它激发人们思考追求知识、幸福与权力的代价，也使人们意识到在不断追求中，灵魂的本质可能会被腐蚀。


《浮士德》是一部让人从视觉到情感都深陷其中的作品，这种沉浸感归功于亚历山大·索科洛夫对文学经典的独特诠释。索科洛夫的《浮士德》并非简单地重述歌德的故事，而是深入挖掘人类对知识和欲望的终极追求，直至其无法承受之重。这部电影更像是一次内心深处的探索——一个在阴暗而扭曲的镜头下的意识之旅，几乎让人喘不过气来。

影片中，索科洛夫通过冷色调的画面和压抑的场景布置，将浮士德博士的心理状态和内心的渴望具象化。视觉上，索科洛夫使用大量的扭曲镜头和暗影，将人物的挣扎和绝望置于一种变形的空间中，仿佛这个世界本身在向我们传达其腐朽和不安。这样的镜头语言不仅仅是表现主义的，还带有一种无形的威压，压迫着观众的情绪。每一个细节都透露出人性深处的欲望和困惑，仿佛在窥探一幅无法言说的心灵图景。

听觉上，索科洛夫采取了一种近乎寂静的方式，背景音乐稀疏且低沉，似乎在刻意回避直接的情感引导，而是让观众在沉默中体会到内心的空虚和矛盾。尤其在浮士德与魔鬼梅菲斯特的对话中，冷静的对白和寂静的背景形成鲜明对比，仿佛一切欲望的声音都在内心深处悄然爆发，而观众只能在这份安静的张力中感受到它们的真实存在。

浮士德与梅菲斯特的关系是影片的核心，他们的契约不仅象征了人类对欲望的妥协，更是对灵魂的拷问。索科洛夫通过对浮士德内心不断加深的绝望与挣扎，揭示了人类永无止境的渴求会带来何等的毁灭。浮士德的欲望并未给他带来真正的满足，反而让他陷入更深的迷失，这是一种精神上的孤独感，如同一种内在的腐蚀。这份腐蚀从视觉、听觉上逐渐渗透到影片的每一个角落，直到观众也无法抗拒这股力量。

在亚历山大·索科洛夫的电影《浮士德》中，批判的核心依然围绕着人类对权力、欲望、和存在意义的无尽追求，但他以独特的方式深化和扭曲了这些主题，将人性中的阴暗面和道德腐化推向极致。索科洛夫的电影语言和歌德的文学表现有所不同，他采用压抑、扭曲的镜头来探索人类精神的脆弱和扭曲。通过视觉的象征性和暗色调的氛围，他不仅是对人类追求的无底深渊进行了描绘，也提出了一种对人性堕落的深刻批判。

具体而言，索科洛夫的批判集中在以下几个方面：
