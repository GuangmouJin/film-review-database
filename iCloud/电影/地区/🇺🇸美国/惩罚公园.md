一、剧情简介

影片采用**伪纪录片（mockumentary）结构**，交叉呈现两条叙事线：

#BFI 长评-希望幻觉。伪纪录片，惊人的批判性，就如同之前看《特权》一样震撼，朋友说过俩月可能有他的回顾展，希望可以看到《巴黎公社》和《爱德华·蒙克》吧！电影以架空的形式映射了麦卡伦国内安全法和当时麦卡锡主义下的社会环境。两种惩罚中“惩罚公园”实际上是一种希望的幻觉，三天徒步是一种希望，结尾处的美国国旗则是讽刺和绝望。双线叙事是把整个“流程”完善，法条和规则实际只是把谋杀合法化，解释权始终在规则制定者手中，“自由”的承诺始终显得虚假荒诞。电影的内核明显是在揭露国家机器的暴力合法化机制，本质跟怀斯曼的电影很类似。

双线模式中，沙漠中我们可以看到被惩罚的人他们的遭遇，我们却不知道他们因为什么走上这条路，而法庭部分则回答了这部分，甚至并不能说回答，而是提供了一套标签化的话语体系，使这些人“被解释”，听证会明显在隐喻麦卡锡那个时段，看似是辩论，本质是早已经预设了结果，而被告和原告间的情绪设计也非常好，一切情绪都被制度和秩序重新编码，暴力命令也被冷静地执行，就跟沙漠中一样，两个都是制度表演的舞台，这也是为什么说跟怀斯曼类似，那一切跟随制度的人，都成为了制度的一部分，不休止的运作。最令人感觉到无奈的是那个小孩射击后不断重复，那是一个意外，但旁边的人也只是告诉他本该如此。

视听上，很喜欢视听上的呈现，不论是手持变焦带来的纪实，还是现场收音的效果。电影中有大量的局部特写镜头，强调的是个体遭遇，面对镜头的访谈…显得极其怪异，一方面纪实性极强的沙漠环境和妆造带来现实感，另一方面直视镜头又在破除电影的幻觉，上次看到这么诡异的感觉还是在瓦尔达的《天涯沦落女》中，现实主义题材结合那种诡异的访谈。电影中有几次比较精彩的设计，暴力执行者用手试图遮住摄影机，或者直接告诉拍摄者停止拍摄，亦或者拍摄者直接参与到争论，感觉这其实是在不断打破“纪录片的真实幻觉”，摄影机不再是中立的记录工具，而是被纳入事件本身，甚至成为暴力结构的一部分。最后就是…作为观众…看这部作品，感觉还是可以找到被批判的感觉，一方面是拍摄者作为记录者似乎也成为了结构的一部分，强调的是媒介的道德，谁在观看？谁在消费（电影中），另一方面则是作为观众观影后的赎罪属性吧，可以避免暴力可能本质是减少暴力传递，但观众确实太安全了（电影外）。


👉 媒体既是“揭露者”，也是“结构的一部分”

具体体现：
	•	摄制组记录暴力
	•	同时又依赖暴力作为内容
	•	在关键时刻试图干预（但失败）

这就产生一个悖论：

👉 **他们是否真的“中立”？**

更进一步：
	•	影片中的媒体
	•	以及拍摄这部电影的摄制组

形成一种镜像关系：

👉 **“谁在观看？谁在制造观看？”**

这就是Watkins最典型的：

让观众意识到：你也是这个系统的一部分

六、恐惧作为社会基础结构（**fear syndrome）**

Watkins自己提到的关键词是：

👉 “fear syndrome”（恐惧症候群）

影片中所有人都被恐惧驱动：
	•	陪审团 → 害怕社会失控
	•	政府 → 害怕革命
	•	警察 → 害怕失败
	•	抗议者 → 害怕被吞噬

但关键在于：

👉 恐惧不是结果，而是**治理工具**

换句话说：
	•	制造恐惧
	•	管理恐惧
	•	利用恐惧

👉 形成一个自我强化的系统


“被要求相信它是假的，同时被说服它是真的”

这正是影片真正的“攻击方式”。

⸻

八、总结（压缩成一句话）

这部电影的内核不是“国家镇压人民”，而是：

👉 **一个通过“规则、影像与恐惧”运作的系统，如何把人从“有语言的主体”，转化为“可被追捕和消耗的对象”。**








三、象征与隐喻

**1. 沙漠空间（Punishment Park）**
	•	极端自然环境 → 社会抛弃之地
	•	无水、无庇护 → 去除文明伪装
👉 象征国家将“异见者”驱逐到社会之外

**2. 美国国旗（终点）**
	•	表面：自由的象征
	•	实质：一个永远无法真正抵达的幻象
👉 爱国主义与暴力结构的悖论

**3. 摄制组（片中纪录者）**
	•	他们既是观察者，也是共谋者
👉 暗示媒体在权力结构中的暧昧位置

**4. 听证会空间**
	•	类似法庭 + 真人秀
👉 语言在这里失效，成为权力的附属物

四、时代背景关联

影片直接诞生于**1960s–70s美国社会动荡时期**：

**1. 越战（**<u>**Vietnam War**</u>**）**
	•	大规模反战运动
	•	青年与政府的冲突激化

**2.** <u>**Kent State shootings**</u>
	•	国民警卫队向学生开枪
👉 影片中的“合法射杀”几乎就是这一事件的延伸想象

**3. 黑人民权运动与激进组织**
	•	如黑豹党（Black Panther Party）
	•	被政府视为“内部威胁”

**4. 国家安全法与监控体系**
	•	《McCarran Act》等法律允许拘留“危险分子”
👉 影片中的制度并非虚构，而是现实的夸张

👉 制度 vs 暴力

一、**Mark Jenkin 评论（当代评论）**

我花了很长时间才意识到，这部电影的声音设计究竟有多么具有实验性。像彼得·沃特金斯的所有作品一样，声音景观在营造那种逐渐蔓延的恐惧与绝望感方面承担了极其重要的作用。由琼·丘吉尔完成的摄影之出色是显而易见的，但由迈克尔·摩尔及其团队完成的现场录音与混音则在不动声色中达到了令人惊叹的程度。

二、**David Wilson（1972，《视与听》）评论（完整直译）**

折页提及，那面星条旗不过是一个虚幻的避难所，而所有这一切，沃特金说，发生在明天，昨天，五年之后——“它同样发生在今天。”《战争游戏》是对不久未来的一种投射，是对即将到来的事物的一种高声警告，而《特权》则是对未来冲击的一种虚构隐喻。在《战争游戏》以及现在的《惩罚公园》中，沃特金呈现出他所谓的“以纪录片形式呈现的寓言”。他因对其电影的“批评性不宽容”而感到愤怒——这种愤怒并非毫无根据——于是他以一种“情感的电影，直接与我们自身对峙的电影”予以反击。对于每一个批评他“歇斯底里”或“偏执”的声音，他回应以“恐惧症候群”，“智性压抑”，“一种我们所有人共有的病态”。事实上《惩罚公园》无疑是沃特金这些“电影小册子”中最好的一部。作为一名电影创作者，如果可以大声呐喊，他就不会选择温和说服，而他也许确实是“偏执”的。

美国代际之间的现实对立被形象地呈现出来。偏执横跨两边：既存在于法庭成员（家庭主妇、参议员、社会学家等）那种焦虑不安的面孔中，也存在于被告那种咆哮、带有自怜意味的固执之中。辩护律师提到希特勒，同时又把他的当事人称为“这些奇怪而被排斥的人”；在法庭午餐间隙对陪审团成员的采访中，可以听到类似“美国需要更多惩戒，少一些斯波克博士式育儿”的言论。追捕小组的队长以冰冷的情绪展示他们的致命武器，他们只是在履行职责，并希望把事情做好，即便这意味着一个满脸青春痘的18岁新兵在镜头前哭泣，说自己的扳机是不小心扣动的。每个人都在重复着中产美国与失意美国那套被条件化的陈词滥调。如果曾经存在过某种沟通的通道，它已经被彻底切断。这一点在一位女性委员对一个女孩的歇斯底里式攻击中表现得尤为清晰（“你是精神分裂，你应该被关起来”），而那个女孩可能正是她的某种映像。在桌子的两侧，人们的脸上写满了恐惧与无法理解，对话早已终结。

有人说，只有暴力才能引发反应。而当一辆汽车收音机接收到尼克松向“面向未来的美国人”发表的讲话时，声轨中却断断续续地混杂着枪声、喷气机的轰鸣以及难以想象的电子噪音。沃特金斯的未来建立在一个“不完美的现在”之上，这既合理也正确。但不那么清晰的是——而正是在这些背景回响之中，疑问开始浮现——这场冲突的具体语境究竟是什么。沙漠中的游戏，就像现实中的游戏一样，并不区分暴力激进分子与和平理性者，但我们却没有得到任何具体情境的说明来解释这种一概而论。更不用说那种令人不安的可能性：如果当前的推测可以成为未来的事实，那么在沃特金斯的逻辑中，任何推测都可以成立；在影片那些令人信服的具体细节背后，却缺乏一种令人信服的“文献基础”。声轨中那种战争般的喧嚣究竟是什么？为什么这些异见者没有意识到“惩罚公园”其实是一个骗局（消息理应早已传开）？如果他们的目标是摧毁美国所代表的一切，为什么还要把希望寄托在美国国旗上？

更严重的问题是：影片接近结尾时突然引入的那个“第三种选择”（大赦承诺）究竟是什么？为什么竟然没有人选择它？而更为严重的是：那些电视摄制组究竟在沙漠中做什么？如果正如采访者在局势失控时对队长所说的那样，这些画面会在NBC播出，那么我们是否应当认为，NBC仍然被允许播出这种显然具有颠覆性的内容？如果不是在现在，那么至少也不可能是在这个未来之中——在那种压制环境下，媒体早已不可能继续作为哪怕是中立的观察者存在。

像以往一样，一旦人们开始质疑沃特金斯设想的整体可信度，这种逼真感就开始动摇。如果这是一则寓言，那么其过于真实的表层反而削弱了人们将其视为寓言的可能性。提出的问题越多，这种可信度的裂缝就越大。我认为，将沃特金斯（正如许多人所做的那样）斥为一种“浪漫的受虐狂”并不成立。他所看到的那种将通向“惩罚公园”的普遍病态，是显而易见的；正如他所说，任何看不见这一点的人，都已经被卷入了这场游戏之中。只是，有些人并不认为沙漠中的一颗子弹就是唯一的终点。沃特金斯说，我们正在失去与现实的联系，这也是为什么他的电影是一则寓言。但他没有看到，或者不愿承认，现实的另一面——那一面或许可以为他的噩梦未来提供另一种可能性。奇怪的是，或许也正是其一贯特征，他并不把这部最具政治性的电影视为一部政治电影。

——David Wilson，《视与听》，1972年春
