# 渣


好的，这部<u>Trash</u>其实很典型地处在“地下电影→被制度吸收”的过渡节点，也正好可以用你常用的六个维度来展开，而且它的很多问题（身体、成瘾、性、资本）其实可以和你之前喜欢的那些“身体/欲望结构”的片子连起来看。


#BFI #35mm 这部作品氛围营造得太好了…类似于John Waters电影中的世界般，整部作品没有特别强烈的叙事，更多的是提供一个视角，边缘群体被投射会被客体化，但影片却用不同的“正常人”来对照呈现边缘人的主体性。而且这个收尾可能是我今年最喜欢的电影收尾了，一方面与影片开场的画面形成一个呼应，仿佛一切都是一个没有尽头的循环（这本身也是半合成物的使用呼应），另一方面在经历这部电影之后，我们似乎可以重新审视整部电影，仿佛经验在此刻被点亮了，一切仿佛与开始又不一样。

角色和隐喻，男主有一半时间都果体，看似与性和半合成物相关，但实际感觉一切却也与性无关，感觉这部分的塑造跟《欲望号快车》很相似，身体成为欲望投射，精致的外貌是天生的，身体的损毁则不光隐喻着越战背景下社会的割裂氛围，也隐喻了资本主义下身体商品化后个体功能的失效，这本身就很讽刺。另一层面养胃也是父权结构的崩塌和男性权力失效的表现。女主Holly本身就是跨性别者，电影中出现的怀孕的妹妹本身感觉也是有对照性，这也促成了和男主的对照关系，她是唯一试图与男主“建立关系”的人，但结果也必然是失败的，但那共存的尝试还是会让人动容。其他角色的状态其实也更加符合美国中产神话破灭，美国梦破碎的表达，富人的性幻想，审查员的恋物，高中生急于证明自己，所有角色都不是在“与Joe他们交流”，而是在“利用他完成自己的幻想”，他更像是一个生产幻想的机器。

折页提及，《渣》的真正主题在开场段落中被清晰地呈现出来，那几乎像是一种形式上的主题陈述。在这一段里Geri Miller尝试用她所能想到的一切方法去激发Joe Dallesandro的性欲望，但后者始终无动于衷，而且并不太在意——无论是她的身体挑逗，还是她精心编排的go-go舞，都无法让他产生反应。Geri以一种近乎母性的方式为Joe感到担忧。她说，问题出在他服用的半合成物上。为什么他不能改为从性中获得“快感”呢？那更便宜，更愉快，也健康得多。Joe问：“你可以从性中获得那种感觉吗？”Geri说：“当然可以”，“当你达到高潮的时候，不是很棒吗？”Joe说：“不是，那就结束了。”

在结尾段落中，莫里西的方法真正显现其意义。这些段落重新汇聚并整合整部影片以惊人的节奏构建出一连串高潮，并最终走向一个反高潮，且过程精准而经济。在其中Joe如同先前所确立的那样，是一个静止的，死亡般的中心，围绕着他运转的是他人的激情。Holly则是动态的元素，尽管她的言行常常极具激烈的喜剧效果，但她逐渐获得了属于自己的尊严。莫里西对她的处理堪称大师级，至于他从她那里所“引导”出来的，究竟在多大程度上可以被称为一种表演，这一点可以无休止地争论，但毫无意义。我们所看到的正如几乎所有电影最终所呈现的那样，是私人梦境的身体化呈现，而这一切之所以能够如此完美地运作正是因为这些人物在其或许疯狂的状态中，被极其谨慎地加以尊重。




在影片的每一个主要段落中，开头提出的主题都以不同的变体被再次呈现。在所有这些段落中，基本情境都体现了“沃霍尔集团”一贯关注的问题，这一点最早在《My Hustler》中得以清晰呈现：即所谓的“交流”，在被仔细检视时，往往不过是个体将自身情感反弹到恰好在场的他人身上。在这里，由于Joe在各个意义上的完全无能（impotence），他成为了他人幻想的共鸣板。有一个疯狂的女人，随身携带一袋玩具，拼命寻找LSD，并毫无根据地坚信Joe一定把它藏在自己身上的某个地方。有一个富有的年轻妻子，在发现Joe试图（但并不十分认真地）盗窃她那几乎空无一物的公寓后，滋生出关于强奸的无望幻想。甚至还有那位来自福利机构的男人，他实际上根本无法与任何人建立联系，只是执着地沉浸于自己关于“琼·克劳馥银色鞋子”的幻想，以及这些鞋子显而易见地可以被改造成一盏时髦而独特的灯。

但最重要的，是Holly Woodlawn。Holly，毋庸置疑，是沃霍尔旗下的变装皇后之一。事实上，这一点甚至无需解释：第一次观看时，人们或许会对她究竟“是什么”感到好奇，但很快就会完全接受她如其所言的身份——一个女人。

她出生时是什么、或者在解剖学意义上仍然是什么，都不重要。她是一个女人，在进行一场表演，而这场表演在任何标准下都具有迷人的力量。除了Joe之外，她是影片中唯一反复出现的角色；Joe与她同住一室，也是她关怀与时常带着恼怒的情感依附的对象。我们看到她在收集毒品垃圾（有时有Joe的帮助，有时没有）；在影片前半段一个既滑稽又悲伤的场景中，她试图勾引一名高中男生，这个男孩急切地想要证明自己完全成熟老练，却愚蠢地认为——

然而，这个角色逐渐发展起来，并在最后的场景中真正获得自身的位置：她姐姐的怀孕让她产生了一个想法——她与Joe可以冒充即将成为父母的人，从福利机构那里获取帮助。不幸的是，有一天她回到家中，发现Joe正试图（当然是徒劳地）与她姐姐发生关系，于是她爆发出一场极为精彩却完全缺乏逻辑的指责场面。随后是与福利官员的一场极其滑稽的场景，最终以她塞在毛衣下的靠垫在一阵激动中掉落而告终。她和Joe最终回到了故事开头时的状态：没有钱、没有前途，甚至在最基本的身体层面上也无法实现交流；然而，无论这种关系的价值有多大或多小，他们仍然彼此相伴。



保罗·莫里西的电影，是一种彻底的人类接受之电影：无论这些角色多么怪异，他们从未被居高临下地对待，从未被嘲弄，也从未被当作廉价的“camp式笑料”。观看的角度，是《肉》与《渣》中最为关键的因素；至于其技巧在某种意义上的惊人表现，则显得相对次要。事实上，它本质上确实是次要的——莫里西属于那一小群导演，他们以一种方式拍摄电影，使得电影本身成为一种透明的外壳，通过它，我们可以以某种心灵感应的方式进入他们的意识之中。

而这种体验，用当下的说法来说，是“震撼心灵的”。

——约翰·罗素·泰勒，《视与听》，1971/72年冬季刊

五、视听语言风格

**1）表演：半即兴**
	•	对话大量即兴
	•	演员像在“聊天”

→ 模糊纪录片/剧情片边界

**2）镜头：极简 + 冷静**
	•	固定机位
	•	长时间停留
	•	很少剪辑节奏变化

→ 一种“观看而不介入”的姿态

→ 现实的“混乱声场”
