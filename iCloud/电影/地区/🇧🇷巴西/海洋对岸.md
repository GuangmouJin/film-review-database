# 海洋对岸

# 
海洋对岸

是<u>沃尔特·塞勒斯</u>与<u>丹妮拉·托马斯</u>联合执导的一部公路式黑白电影，也是巴西“后Cinema Novo时代”中极具代表性的作品之一。


#BFI 长评-海的那边是什么？同步了冷战后期各个夹缝中国家电影浪潮的视角，那种漂泊，亦或者这已经不能算是漂泊了…更像是“无家可归”。故事设定在1990年巴西经济崩溃时期，这种“无家”的状态本身也是对于当时巴西现实的一种私人化的回应。影片中讨论了很多，但感觉内核都是何去何从，种族，殖民，经济废墟，自我认同…似曾相识。而从开始母亲向往的回归到结尾处男女主的情感都仿佛梦幻泡影，所以回到标题的一个疑问，答案似乎是海的那边什么也没有。

角色和隐喻，在巴西的男主和在葡萄牙的巴西裔女主本身是一种呼应，男主虽然离开了巴西，但威胁却一直存在，而女主虽然生活在葡萄牙却始终带着巴西移民的身份阴影，角色与空间之间仿佛带着安东尼奥尼式的疏离。同时特别喜欢空间的这部分设计，影片没有把葡萄牙设计成一个理想的避难所，甚至当镜头转到葡萄牙没有什么不同，同样的贫穷和犯罪，同样破败和迷失，片中的“国”不仅仅是地理上的表达，更是一种人在“家”与文化中的精神状态。

财富与掠夺，影片还提到了“阿基尔”，就是赫尔佐格那部电影中的描述的阿基尔，寻找黄金国，失败似乎也是某种无法逃离的隐喻，后续的台词中所强调劳动力则是一种随时代替换的被剥削物，当年的黄金，后来的蔗糖，橡胶和钻石，现在的则是“额头的汗水”。偷运小提琴里面藏着钻石很具有讽刺意味，小提琴本身就具有一种精英，阶级或者文明象征，而与男主偷运的事件放到一起就像是在象征殖民者对于被殖民者的资源掠夺，所谓“文明”背后就伴随着资本与暴力，结局的无人在意也是最好的讽刺。

视听上，很有特色，也很熟悉，不少的场景让我想到了《怒火青春》，母亲去世后那段手持非常喜欢，尤其是无序抖动搭配节奏感极强的配乐和户外的高对比黑白。影片中大量的近景和特写，感觉不光是对于角色情绪和情感的呈现效果极好，还可以直接从视觉上提供一种压迫感。后配音的音色差有些怪，不喜欢，配乐非常喜欢，尤其是那种带有民族风格的配乐。

总结

《海洋对岸》是一部关于：

* 国家崩塌后的身份危机
* 全球化下的流亡感
* 后殖民历史循环
* 90年代第三世界幻灭

的电影。

它最迷人的地方在于：

角色似乎始终在移动，但这种移动并不意味着自由，而更像一种无法停止的漂流。

所谓“海洋对岸”，最终只是一个永远无法真正抵达的想象。
