# 国家公诉人


🎬**《国家公诉人》（Zwei Staatsanwälte, 2025）**

#ICA **中规中矩，平淡且简单，感觉这种叙事讲述那个时期有些过于直白，没有讽刺，只有一个批判。远景大远景效果还行，大量的固定镜头批判性强，尤其是那空间压迫感也很足，就是……干巴。信贯穿，单没意义，没有附加任何其他。**


**2）内核表达**

洛兹尼察一贯关注“权力如何通过形式维持自身”，本片延续其对极权结构的冷静剖析：
	•	**司法作为表演**：审判并非寻找事实，而是再生产权力的合法性
	•	**个体伦理 vs 体制逻辑**：当个体意识到荒谬时，选择沉默还是背叛？
	•	**语言的空洞化**：法律语言被彻底去意义化，只剩下形式的重复

影片的核心不是“冤案”，而是**冤案如何被系统性制造并合理化**。

⸻

**3）象征 / 隐喻**
	•	**“密函”**：
象征被压抑的真实，同时也是体制内部无法彻底清除的裂痕
	•	**双公诉人结构**：
并非简单的善恶对立，而是同一系统中“信仰阶段”的差异
→ 一个代表“尚未怀疑”，一个代表“已经觉醒但无力行动”
	•	**审判空间**：
法庭更接近剧场（theatre），观众、演员、文本高度固定
→ 强烈呼应“仪式化政治”的概念

⸻

**4）时代背景关联**

影片显然指向**斯大林大清洗时期（1930s）**，但洛兹尼察的处理并不历史再现式，而是抽象化的：
	•	对应历史事件：<u>斯大林大清洗</u>
	•	“认罪文化”与“公开审判”
	•	国家通过司法系统生产“敌人”

但更重要的是，它并不局限于苏联历史，而是具有普遍性：

👉 **任何高度官僚化的权力结构，都可能滑向这种“自我证明式审判”**

⸻

**5）视听语言风格**

这部分是洛兹尼察最关键的表达方式：

**① 长镜头 + 固定机位**
	•	几乎“非介入式”的观察
	•	观众被迫像“旁听者”一样进入空间

**② 去情绪化表演**
	•	演员几乎不提供心理外化
	•	强调“角色作为功能单位”而非人物

**③ 声音设计**
	•	极度克制的配乐（甚至无配乐）
	•	以环境声和语言重复制造压迫感

👉 听觉上形成一种“制度噪音”：
不是音乐推动情绪，而是**语言本身变成压迫机制**

**④ 空间调度**
	•	门、走廊、办公室反复出现
	•	强调权力的层级与不可穿透性

⸻

**6）风格来源 / 影响 / 是否改编**

✔ **作者延续**

与洛兹尼察之前作品高度一致，例如：
	•	<u>国家葬礼</u>
	•	<u>审判</u>

都在探讨“历史影像 / 仪式 / 权力结构”的关系

⸻

✔ **形式来源**
	•	**苏联纪录影像传统（反讽性继承）**
	•	**布列松式去心理化表演**（你之前也常提到）
	•	**戏剧化结构（但反戏剧）**

👉 有点像你之前分析里维特时说的那种：
**“戏剧被强化成仪式，而仪式本身暴露其虚假性”**

⸻

✔ **是否改编**

本片并非直接改编具体文学作品，但明显借鉴：
	•	苏联审判档案
	•	历史记录文本
	•	政治演说语言

是一种“文献性再构”（documentary fiction）

⸻

🧩 **简单总结（给你风格向）**

这部片其实可以这样抓核心：

它不是在讲一个案件，而是在拍“案件如何被制造”。

或者更进一步：

审判不是为了裁决，而是为了让权力被看见、被重复、被相信。
