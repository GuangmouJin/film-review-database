# 情感价值


#ICA 

戏中戏 拍摄 电影更像是家庭 而家庭成了剥削的对象 想起了 Mai的儿子当时的介绍


2星

#ICA 这个片子看着非常眼熟，形式上感觉跟《哈姆奈特》很相似，讨论的是创伤与艺术性的转换，内核又接近于伯格曼那种创伤的代际遗传，从战争到战后再到现代，而视角则是以“屋”为主视角展开，那苍老的旁白声仿佛就是屋子的化身。男主创作的剧本看似是替换女儿的情绪，但实际也带入了自己的情绪，对应女儿的脆弱他知道却不退让，而自己的脆弱清楚又也逃避，感觉怪怪的，这种代际裂痕感觉让我想到了**安格拉·夏娜莱克**的作品，不过可能那种间离的呈现我会更喜欢，这种不得罪人的手法感觉个他上一部作品很类似，就是不够纯粹，什么都想讲，物品的价值，回忆的价值，现实的价值，戏剧的价值，电影的价值，艺术的价值，最后结尾也不咋喜欢，离开了屋子，但仍回到电影，仿佛看《塔尔》……




影片反复使用**片中片 / 电影计划 / 未完成的作品**作为隐喻：
	•	情感被剪辑
	•	记忆被取景
	•	他人的人生成为叙事素材

这与你熟悉的**元电影传统**（里维特 / 戈达尔 / 伯格曼晚期）高度同频。

3️⃣ **家庭 = 创作现场**

家庭空间不再是私密之所，而像一个：

**永远处在筹备期、排练期、却从未正式完成的片场**

⸻

四、时代背景与文化语境

虽然是当代挪威电影，但它明显处在一个**更广义的北欧—欧洲作者电影脉络**中：
	•	**后福利社会的情感空心化**
物质保障稳定，但亲密关系却极度脆弱、疏离。
	•	**“男性天才作者”的清算阶段**
斯卡斯加德饰演的父亲，几乎是一个被时代追问的形象：
你的一切伟大，是否建立在对他人情感的透支之上？
	•	与《世界上最糟糕的人》不同，这次提尔几乎**完全放弃浪漫化**，转向冷静、甚至残酷的伦理层面。

⸻

五、视听语言风格（比前作更**“老派”，也更锋利）**

🎥 **视觉**
	•	稳定构图、克制调度
	•	家庭空间多为**封闭、低饱和、缺乏纵深**
	•	人物经常被框在门框、窗框、影像边缘中——
👉 **视觉上始终处在“未进入”的状态**

🔊 **听觉**
	•	对话极其重要，但**大量关键情绪发生在“说不出口”的停顿中**
	•	音乐使用非常节制，更像情绪的残影而非推动力
	•	有明显的“后配感”，强化一种**回忆被重新讲述**的质感

✂️ **剪辑**
	•	不追求情绪高潮
	•	情感节点往往被**提前或延后切断**
	•	让观众处在一种：
“我理解了，但我不被允许释放”的状态

⸻

六、风格来源 **/ 影响 / 是否改编**

🎞 **风格来源**
	•	**伯格曼晚期家庭电影**（《芬妮与亚历山大》《婚姻生活》）
	•	**里维特的家庭—剧场结构**
	•	**沃格特一贯的心理写实主义**
	•	对“作者神话”的反思，明显延续自**《奥斯陆三部曲》但更冷**

📚 **是否改编**
	•	**原创剧本**
	•	但在结构和伦理问题上，明显吸收了：
	•	北欧戏剧传统（易卜生式的家庭道德困局）
	•	当代作者电影对“自我暴露”的反思

⸻

一个总结性的判断（偏影评立场）

如果说《世界上最糟糕的人》还保留着对情感流动的浪漫信念，
那么 **《情感价值》几乎是在问：**

**当一切情感都被叙事化、艺术化、心理化之后，
我们是否还真的“欠彼此什么”？**

这是一部**不提供抚慰的家庭电影**，
更像一次对“情感是否还能成立”的冷静审计。
