# 

先直接说结论：**这部片（尤其你贴的这一大段）不是在“讲故事”，而是在用一串互相矛盾的证词和宣传影像，拆穿“全民抵抗”的神话，逼观众面对“合作”“冷漠”“搭便车”这些灰色地带。**

⸻

**1. 简要剧情 & 结构（整体上先放在心里）**

《悲哀与怜悯》副标题就是：**“一座被占领城市的编年史”**，拍的是中部城市克莱蒙费朗在德军占领、维希政权统治时期的全貌：
既有抵抗组织、戴高乐派，也有合作者、普通市民，还有德军军官、纳粹官员。影片由大量**访谈 + 战时新闻纪录片 + 纳粹 / 维希新闻短片**拼接而成。 

通常被分成两部分：
	•	**Part 1：The Collapse（失败）**——军事失败、停战、维希政权成立；
	•	**Part 2：The Choice（选择）**——合作、抵抗、冷漠之间的取舍。 

你给的这一大段（1070 以后）大致处在：

占领秩序已经建立 → 维希政权的“社会革命”宣传 → 日常生活的匮乏与变态常态 → 系统性的反犹与合作 → 早期抵抗逐渐成形。

⸻

**2. 你贴的第一部分台词，大概在表达什么？**

我按几个小板块来拆：

**2.1 维希政权的“社会革命”话语**

“他认为我们的社会制度不公平……他决心给法国带来幸福……这就是他的社会革命。”（1070 以后那段）

这里是**维希青年官员的宣传口吻**：把佩当的“国家革命”（工作 / 家庭 / 祖国）包装成一种温柔的“社会改革”“消除贫困与不公”。

奥菲尔斯的剪辑手法是：
	•	先让这种**甜言蜜语式的话语**完整地说出来；
	•	后面马上跟上**反犹法令、教师被解雇、小商贩登广告“100%纯法国人”**、犹太药剂师助理被孤立…
→ 让观众自己感受到：所谓“社会革命”，真实的内容其实是**种族清洗、排斥、服从秩序**。

**2.2 个人小叙述里，如何为自己开脱？**

Mendès-France 越狱那段非常典型：
	•	他讲自己如何在墙上犹豫、听到树下情侣谈恋爱的声音、等女方终于答应才跳下去。
	•	这段听上去近乎幽默、浪漫，但背景是：**他作为犹太议员，被维希以“逃兵”罪审判、关押。**

奥菲尔斯在这里做的，是用一种**“轻描淡写的个人趣闻”**，夹在极其沉重的政治迫害里：
	•	一边是男人为了性、女人为了“道德”犹豫；
	•	另一边是政治犯在生死边缘踌躇。

这既显出 Mendès-France 的机智、幽默，也透露出一丝**残酷讽刺**：

在历史灾难中，私人爱情、欲望仍然在运转，它既可成为生还的契机，又显得格外不相称。

**2.3 匮乏与“涂丝袜”：日常生活如何被卷入**

没皮鞋、没火柴、什么都缺，然后是“没有丝袜的丝袜”：涂腿、画丝袜缝线、还防水……

这一组新闻片段看上去像生活方式节目，但放在整体里很尖锐：
	•	一方面呈现被占领下的**经济匮乏、物资短缺**；
	•	另一方面，女性身体被要求继续维持“优雅”“好看”，于是出现这种荒诞的替代物。

这里的重点不是“搞笑”，而是：
	•	维希政权和德国宣传不断要人们相信：**“秩序恢复了，生活还在继续，甚至更健康、更有建设性。”**
	•	奥菲尔斯把这些影像放在**反犹、审判、逃亡**的叙述之间，形成强烈错位——
→ 观众意识到：**“正常生活”本身就是宣传的一部分。**

**2.4 反犹机构、展览与“小人物”的反应**

你贴到的中段有好几条线交织：
	•	“犹太问题研究所”纪念反犹作家德鲁蒙，展览《犹太人与法国》、测量脸型、讲解“混种种族特征”；
	•	当地老师说：
	•	自己有朋友因为母亲是犹太人无法再任教；
	•	学校里犹太老师被开除；
	•	同事口头上“同情”，但没人集体辞职。
	•	小店主 Klein 被怀疑是犹太人，于是登广告“100%纯法国人”，强调自己兄弟在一战中的军功。

这一整串，其实就是把“结构性反犹”拆成几个层次：
	1.	**上层制度**：法令、机构、展览、伪科学。
	2.	**中层专业人士**：教师、药剂师、检查员，在“遵守规定 + 个人同情”之间摇摆，最后大多顺从。
	3.	**小人物**：害怕自己被当成犹太人，于是通过军功、广告去证明“纯法国性”。

你能看见一种反复出现的姿态：

“我个人不种族主义，可是……我没办法 / 我也得活 / 我只是想自保。”

奥菲尔斯并不直接指责，而是让这些人自己把这套逻辑说出来；**其残酷恰恰在于他们都觉得自己“不是坏人”。**

**2.5 电影、审判与宣传：一种元电影层面**

这一部分大概是整部片里最“meta”的段落之一：
	•	一方面，Mendès-France 说自己常去电影院，因为：
“那里黑、舒服、没人看到你。”
电影院成为**逃亡者的庇护所**。
	•	另一方面，影片立刻插入纳粹反犹宣传片《犹太人苏斯》的片段：
	•	庭审、绞刑、煽动仇恨的对白；
	•	法官引用“古老法律”“犹太人与基督徒通奸应当绞死”。
	•	接着是旁白式访谈：观众起初以为只是普通德国片，很快意识到是恶毒宣传，大量观众开始“观影罢工”。

奥菲尔斯通过这种剪辑，让你看到：
	•	**影院既是猎手的工具，也是猎物藏身的地方；**
	•	战时电影工业如何服务于**仇恨建构**；
	•	而二十多年后的他，又用电影（纪录片）把这些影像重新剪开、拼接、反讽。

这就是你喜欢的“元电影味”：电影中有人说“我用电影躲避”，同时我们正在看一部用旧电影拆穿这套机制的新电影。

**2.6 巴黎的两面：享乐与挣扎并存**

高层观众、夜总会、Maxim’s、Le Boeuf sur le Toit 生意兴隆；
有人说：“当时的巴黎其实非常好玩、疯狂。”

这一段是对战时巴黎**“派对一面”**的直视：
	•	一边是饥饿、配给、迫害；
	•	另一边是上流社会继续狂欢、电影工业兴旺，甚至因为部分犹太制片人逃往美国，**某些法国电影质量还更好了**。

奥菲尔斯把这些“回忆中的轻松语气”保留下来，不用旁白斥责，而是让它自己暴露一种**令人不适的轻浮与失忆**。

**2.7 学生、教师与抵抗：记忆空白与纪念牌**

后半段回到克莱蒙费朗高中：
	•	教师谈到一些学生进入抵抗组织、被捕、死后如今有街道以他们命名；
	•	当被问起：“当时同学怎样反应？有同学父母被捕，第二天来上课，大家怎么对他？”
→ 他们统一回答：“我想不起来了。”

镜头随后切到**校内纪念墙**——二战阵亡学生的名字。

这里非常冷：
	•	公共空间有碑、有“纪念”，
	•	但当年现场的情感和反应，却被“记不起”抹平。

这跟片名的“悲哀与怜悯”呼应：
	•	**悲哀**在于记得的人太少；
	•	**怜悯**常常是事后姿态，不能抹去当年的冷漠和顺从。

⸻

**3. 内核表达：它真正想拆掉的是什么？**

概括几条：
	1.	**打破“全民抵抗”的神话**
片子系统地呈现：
	•	有抵抗者，但人数有限；
	•	有大量的观望、机会主义、怯懦；
	•	有真心的反犹和合作，也有为了自保的配合。
它直接挑战战后戴高乐式的“整个民族都在抵抗”的官方叙事。 
	2.	**责任与选择的灰区**
很多人都说：“那时候也没办法”“我们尽量帮一点忙”——
奥菲尔斯不是要简单地说“你们都是坏人”，而是逼观众承认：
大部分人都在**夹缝中做了对自己最安全的选择**，而这些累加起来，成了一整套犯罪机器的一部分。
	3.	**反犹与英法情结的交织**
台词里反复提到：
	•	对英格兰的怨恨（敦刻尔克、拒绝停战）；
	•	把犹太人、共党、共济会、外来者打包成替罪羊。
这种情绪本来就存在，被失败与宣传一激，就再度浮上台面。 
	4.	**记忆、遗忘与“事后英雄主义”**
许多受访者显然在为自己当年的行为“整理叙事”，
奥菲尔斯靠**追问 + 对照影像**不断打破这种自我粉饰，提出问题：
我们今天讲的“历史”，到底掩盖了多少当年的自私和恐惧？

⸻

**4. 象征 / 隐喻（在纪录片里的那种）**

虽然是纪录片，但这一部分象征味道其实很强，只不过是“现实中的象征”：
	•	**“给我你的表，我给你时间”：**
民间关于“合作”的笑话，本身就是**极精准的政治隐喻**：
	•	合作为谁服务？
	•	谁交出资源，谁掌握话语权？
	•	**涂丝袜的腿**：
	•	象征匮乏下“伪装正常”的欲望；
	•	也暗示“文明”和“体面”其实是一层可以洗掉的颜料。
	•	**电影院**：
	•	对犹太人来说是**藏身之所**；
	•	对纳粹来说是**洗脑工具**；
	•	对奥菲尔斯来说，是**重新剪辑记忆的战场**。
	•	**老师们的“记不得了”**：
	•	是一种**心理防御机制**的写实呈现；
	•	也像一种集体的“历史失忆症”，本身就成了隐喻。
	•	**佩当与儿童、三联画“工作 / 家庭 / 祖国”**：
	•	把国家的未来投射到孩子身上，要求他们“重建法国”；
	•	实则是用**父权式温情**掩盖政权的压迫与排斥。

⸻

**5. 时代背景：为什么 1969 年拍这个这么“炸裂”**
	•	拍摄时间：1969 年，离战争结束不过 25 年，很多人还活着。
	•	法国战后很久都以“受害者 / 抵抗者”的形象看待自己，官方电视台 ORTF 委托后又拒绝播出这片子，它被法国电视封杀了 10 多年，要到 1981 年才在电视首播。 
	•	这也跟 60 年代末的气氛有关：阿尔及利亚战争记忆、新左派运动、对国家暴力和殖民历史的反思，都在推动重新审视二战时期的行为。

所以这部片既是**关于 1940s 的历史片**，也是一个**1960s 的政治事件**。

⸻

**6. 视听语言风格、影响、是否改编**

**6.1 视听语言**

**形式上：**
	•	**大量“坐着说话”的长访谈**（政治人物、农民、小商贩、教师、抵抗者、德军军官）；
	•	穿插**战时新闻短片、宣传片、纳粹电影片段**；
	•	几乎没有“上帝视角”的解说，而是靠**剪辑把不同观点对撞**，营造反讽；
	•	节奏上允许**冗长、结巴、沉默**，不怕无聊，让观众直接面对这些人的语气、眼神。 

**这和你熟悉的：**
	•	**Resnais《夜与雾》**的冷静档案式对照；
	•	**Rouch / Morin《夏日纪事》**那种“人自己说出矛盾”的 cinéma vérité；
	•	再加上奥菲尔斯特有的**“残酷幽默感”**（比如香槟、丝袜、晚餐闲聊）

有点像**直面现实的口述史 + 马克思式讽刺剪辑**。

**6.2 可能的影响来源**

学界一般会提到： 
	•	战后欧洲“**记忆纪录片**”传统：Resnais、Marker；
	•	美国那边的 **direct cinema / cinéma vérité** 对“抓原话”的影响；
	•	但奥菲尔斯的剪辑并不假装中立，而是很明确地**通过结构来“控诉”和质询”**。

你会感觉他有点像 Wiseman + Resnais + 一个毒舌历史学家。

**6.3 是否改编？**

不是改编作品。
	•	这是奥菲尔斯受电视台委托，自己做的**原创长篇纪录片**；
	•	但使用了大量战时的现成素材（新闻片、宣传片、电影），有点接近“**档案片 / found footage**”的方式。 

⸻

如果你之后想写这部片的影评，我觉得可以从这几个切口入手：
	1.	**“合作”这条线：**从政治高层到小人物自保；
	2.	**电影 / 宣传与再回收：**《犹太人苏斯》片段 + Mendès-France 躲在影院 + 奥菲尔斯的再剪辑；
	3.	**记忆空白与“事后英雄主义”：**老师“记不得了”、纪念牌、佩当与孩子；
	4.	和你正在研究的安哲《雾中风景》《养蜂人》的“国家记忆 / 失忆”放在一起对照，应该会很好玩。
