# 《扎齐坐地铁》

# 

**《**<u>**[扎齐坐地铁](chatgpt://generic-entity?number=0)**</u>**》**

Zazie dans le métro，1960，路易·马勒



#PrimeVideo 以小孩子的视角反馈成人世界的结构性问题，这个视角很多作品共都有类似的呈现，地铁作为驱动其实本身是底层所面临的需求，而开场因罢工而关闭的地铁与呈现的是当时的社会状态，儿童作为去社会化的形象则是反馈了社会对她的教化。片中大人所有情绪行为都被放大了，形成了一种极强的讽刺性，同时那种荒诞又不会显得很严肃，感觉跟我看过的导演的另几部作品还是有很大分别的。

风格与隐喻上，风格最明显的可能还是接近于塔蒂的那种讽刺和卓别林的闹剧质感。于洛和扎齐是一种类似的去社会化的角色设计，不过性格和影片的讽刺方向上不太一样，塔蒂的角色是被迫进入现代社会，但却于社会格格不入，而现代社会被描述成看似高效实则低效的笨拙状态。本片中的现代社会是荒诞的，而扎齐（去社会化）是拒绝了解现实社会的，但即使她不断逃离成人的控制系统，却始终被更大的社会结构包围，呈现的是即使社会只剩形式在运转仍然具有不可超越性。舅舅的状态则像是现代社会下对于个体异化出的不同面具，舞台和表演非常强是用来间离呈现结构性问题，片中有一幕是用装修来强调时代变化，而家庭和国家的完整性（维希将军的照片与宪兵的突袭在结尾）仍然在运作，那一切去政治化仿佛都是假象，权威结构早已经自然化和日常化。

视听效果还是很统一的，大量的快剪，跳剪，跳轴和重复，不光是有那种跳脱和欢愉的小孩视角，还带有强间离的呈现，还有那种类似音乐剧般的台词形式和大量的噪音，感觉也都是为了制造间离让观众避免沉浸，像是一种反音乐传递，反童话美好，反叙事的表达，仍然强调那些是对观众的操控。






改编自雷蒙·格诺（Raymond Queneau）的小说，本身就属于：
	•	语言实验传统
	•	乌力波（Oulipo）前史
	•	对“标准法语”的拆解与恶搞

马勒不是在“忠实改编”，而是在**把文学语言的暴力转译为电影形式的暴力**。
