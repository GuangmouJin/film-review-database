# 蛇蝎美人


**Blonde Cobra，美国，1963，33分钟，16毫米**

#ICA #16mm 电影依然没有完整的剧情，影片是由Jack Smith的录音和Bob Fleischner拍摄的未完成影像拼贴而成。已经能看到Jack Smith的作品风格了，感觉很像最近刚看的《正常的爱》，混乱仪式的演绎这部分蛮喜欢的，那种轻盈生活状态的“完全”呈现，挑剪呈现出的效果与完全的音画分离也是比较喜欢的部分，电影放映途中暂停了一下，说是音源需要调整，放映期间的背后播放的广播填补空白非常有趣，感觉像是一种跨时代的对话。而且声音作为更强的部分呈现，比影像更有力量，更多的是以影像迎接语言。

Ken Jacobs：“《蛇蝎美人》是一种不稳定的叙事——不，其实并不是叙事，它只是为了传递的方便而在时间中被拉长。这是对一种爆裂生命的窥视，是对一个充满想象力之人的窥视，他在尚未被时尚化的下东区贫困中受苦，同时被对美国50年代，40年代和30年代的厌恶所吞噬。愚蠢，自怜，被罪疚束缚，但又取得胜利。”

嘉宾**Mark Webber映前**介绍， Bob Fleischner放弃了他原本在拍摄的素材——那是关于Jack Smith和Jerry Sims的，他们原本想拍一部恐怖片，但没能完成。他们最终闹翻了，Ken得到了这些素材并创造出了一个全新的世界，他把它们和音乐以及Jack Smith的录音结合在一起，那些录音是Jack即兴讲述故事，重复他在日常生活中随口说出的句子。比如“性就是一件烦人的事”，以及“既然我连活着的理由都想不出来为什么还要刮胡子？”尽管这部电影的拼接方式在当时是非常激进的，但今天看来它似乎已经不再那么具有冲击力——不像它刚上映时，面对普通观众时那样，但“震撼”并不是重点。他们只是打扮，讲故事，沉浸在一种夜莺般的幻想之中。Jonas Mekas曾提出“**Baudelairean cinema**”这个概念，用来描述地下电影中的一个子类型，而Jack确实在片中引用了<u>Charles Baudelaire</u>的话：“生活充满了天真无辜的怪物”，这种在日常生活中同时存在的美与恐惧的观念，贯穿了这些作品，也贯穿了Ken的全部创作，一直到他晚年所做的那些作品。




他是一个完全独特的艺术家和人格，我希望你们能够继续关注接下来的放映。

并且去体验他的作品，以及通过他的文字所传达出来的东西——这些文字也被印在书的后面。




👉 **“人格爆炸的影像化”**

Jacobs的描述很关键：

not really a narrative… an exploding life

核心主题：
	•	贫困与欲望
	•	自我厌恶与自我神化
	•	美国文化的“垃圾审美”

👉 Jack Smith =
**边缘人格 + 表演人格 + 文化废墟中的主体**

⸻

**3）象征 / 隐喻**
	•	“Cobra” → 危险 / 性感 / 神经质人格
	•	声音先于影像 → 主体不是身体，而是意识流
	•	影像碎片 → 人格碎裂

👉 可以理解为：
**前心理惊悚 / 前自传电影**

⸻

**4）时代背景**
	•	Lower East Side贫困艺术家群体
	•	同性恋地下文化（当时仍高度压抑）
	•	与 <u>Flaming Creatures</u> 同一文化语境（被查禁）

👉 这是一个**非法、边缘、反体制的艺术空间**

⸻

**5）视听语言风格**

**视觉：**
	•	found footage + unfinished footage
	•	强烈的拼贴、断裂
	•	非连续性

**声音：**
	•	非同步录音
	•	Jack Smith独白（极具表演性）

👉 声音 ≠ 配音
👉 声音 = 主体

⸻

**6）风格来源 / 影响**
	•	<u>Jean Cocteau</u>（自我神话）
	•	<u>Charles Baudelaire</u>（堕落美学，Mekas所谓“Baudelairean cinema”）

影响后世：
	•	<u>John Waters</u>
	•	<u>Andy Warhol</u>

👉 可以说这是**酷儿影像 + 垃圾美学的源头之一**
