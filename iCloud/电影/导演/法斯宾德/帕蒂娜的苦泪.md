# 帕蒂娜的苦泪



#PCC 长评-爱情是暴力与幻象。电影的故事情节，角色塑造，视听语言和内核表达都非常喜欢。内核是权力的游戏和孤独的宿命，在我看来爱情不过是幻象，电影很大程度是在解构爱情，想到齐泽克说的“爱情是一种极端的暴力行为。”女主也只是自我摧毁，就像影片中女主没有理性地爱Karin，而是突然陷入一场毁灭性的爱中，Karin更多的是一个符号，可以是任何人，我感觉所有的角色都知道自己需要什么…之才是最吸引人的，他们都带着“面具”而已。女主知道她只是想要控制，没有真正的爱，Karin知道她也只是一个物品会用尽可能快的方式拿到自己想要的并逃离。Marlene我认为是最特殊的，她不是参与者，更像是一个观察者，但是当被邀请进入这场结果注定只能逃离的游戏时，她选择了提前离场。镜头调度，构图，色彩实在是太精彩了，现场收音的效果极佳。



#PCC 我就知道法斯宾德不会让我失望，他仍然是我目前最喜欢的导演。电影的故事情节，角色塑造，视听语言和内核表达都非常喜欢。

电影的内核与其说是爱与孤独不如说是权力的游戏和孤独的宿命，在我看来爱情不过是幻象，电影很大程度是在解构爱情，想到齐泽克说的“爱情是一种极端的暴力行为。它并不是‘我爱所有人’，而是‘我选择了某个特定的人，并对他说：我比任何东西都更爱你。’ 在这种严格的形式意义上，爱情是邪恶的。”当这种灾难持续出现时，也只是一次次的自我摧毁罢了，就像影片中呈现的女主没有理性地爱Karin，而是突然陷入一场毁灭性的爱中，这里的Karin更多的是一个符号，可以是任何人，而且就像Karin同款的娃娃在影片中也存在其他的娃娃，仿佛也只有Marlene知道这三年有多少个Karin。

影片中每个角色都真的不知道他们需要什么吗？其实我感觉所有的角色都知道自己需要什么…之才是最吸引人的，他们都带着“面具”而已。女主知道她只是想要控制，没有真正的爱，这个角色的很明显是法斯宾德自身的投射，具有极强的情感依赖需求和对于失去控制的恐惧，她在某系方面是极度自卑的，但是又在另一些方面极度自恋，法斯宾德的作品中有大量的角色的性格是这样的设定。Karin知道她也只是一个物品，她知道如何利用女主的欲望，她对于这个房间没有留恋，用尽可能快的方式拿到自己想要的并逃离。Marlene我认为是最特殊的，她不是参与者，更像是一个观察者，她清楚的知道女主的曾经，也了解这三年的经历，她有爱吗？感觉是有，但是当被邀请进入这场结果注定只能逃离的游戏时，她选择了提前离场。

孤独呢？其实感觉是一种宿命，因为他们都戴着面具，所以一切都如同一个剧本般，他们的结局早已经写好了，电影的几个假人的状态摆放也很大程度地反映了角色们的关系状态。

视听，最喜欢法斯宾德的原因就是精彩的视听呈现，一个空间内完成的一部2小时的电影，让你对这个空间和空间内的事物一直保持新鲜感是需要精致的调度，长镜头的使用和角色的调度实在是太精彩了，长镜头可以帮助铺垫角色情绪，考究且有表达的构图也是一样，变焦镜头则是帮助呈现每个角色的情绪变化。Marlene这个角色以各种形式存在在电影中，在画面的后景亦或者打字机的声音，这些提醒着观众她的“在场”，这个角色其实更像是“观众”，看着女主和其他角色表演。现场收音的效果极佳，完全没有银色差，而且现场的黑胶机的播放也很符合角色表达。

有两场戏印象比较深，第一场戏是从女主摇镜头到Marlene后女主的电话内容让Marlene虚焦到模糊持续一阵儿后再摇回到女主的脸部特写。第二场则是女主与Karin在画面中的构图与后景的画作完全呼应的站位，呈现出了权力关系。

如何与《佩特拉·冯·康特的苦泪》联系？
	1.	佩特拉的爱情是灾难性的（catastrophic love）
	•	佩特拉并没有理性地爱Karin，而是突然陷入一场“毁灭性的爱”。
	•	她的“我爱你”并不是温和的情感，而是一种彻底的投降，是对自我的消解。
	•	这正是齐泽克所说的：“爱是一场灾难，它摧毁你的完整性。”
	2.	佩特拉的爱是排他性的（exclusive love）
	•	她不可能对所有人都好，她只想要Karin，并要求Karin完全属于她。
	•	这与齐泽克说的“爱不是‘我爱所有人’，而是‘我爱你胜过一切’”的观点一致。
	•	这种排他性使得爱情成为一种暴力的权力斗争，最终导致关系破裂。
	3.	爱情的暴力在于它的颠覆性（disruptive love）
	•	佩特拉原本是一个成功的女性，拥有事业和地位，但爱情使她彻底崩溃。
	•	她从掌控者变成了受害者，这种身份的剧烈变化正是齐泽克所说的“爱情的暴力性”——它颠覆了一个人的稳定状态，让她变得脆弱、痛苦、失控。
	4.	Marlene的离开是对“爱情暴力”的拒绝
	•	Marlene始终是旁观者，她没有卷入佩特拉的情感漩涡。
	•	最后，她带走象征Karin的娃娃，却选择离开，这意味着她拒绝成为这场爱情暴力的受害者。
	•	这与齐泽克的观点一致：真正能够掌控自己的人，是那些不被爱吞噬的人。

总结

齐泽克的“爱情是暴力”观点，强调爱情的三重矛盾：
	1.	爱情颠覆自我，让人失控 → 佩特拉在爱情中失去自己，变得脆弱而疯狂。
	2.	爱情要求排他性，导致权力斗争 → 佩特拉试图控制Karin，但最终被对方反过来支配。
	3.	爱情是一种创伤性的事件（traumatic event） → 佩特拉的爱是痛苦的，它不是治愈，而是自毁。

**《佩特拉·冯·康特的苦泪》可以说是对齐泽克“爱情是暴力”理论的绝佳例证。**佩特拉的结局验证了他的观点：爱情不是温柔的、幸福的，而是一种极端的、不稳定的、甚至自我毁灭的体验。
