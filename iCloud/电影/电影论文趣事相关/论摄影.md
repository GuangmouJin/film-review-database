# 论摄影


苏珊·桑塔格在《论摄影》中提及惠特曼式的摄影理念，主要是指美国诗人沃尔特·惠特曼（Walt Whitman）在其诗歌中所倡导的一种包容性和民主性的世界观。惠特曼的作品尤其是《草叶集》中，通过对多样性、普通人和日常生活的歌颂，表达了一种万物平等和美好共存的理念。在摄影领域，这种“惠特曼式”的理念体现在摄影试图通过捕捉一切事物——无论是美的还是丑的、高尚的还是卑微的——来呈现一个完整、无偏见的世界。

摄影实践中的惠特曼式理念在以下几个方面体现：
	1.	平等性：摄影不区分高低贵贱，它捕捉到的是现实中的每一个细节，任何事物都有其存在的价值。
	2.	包容性：摄影镜头对世界的关注是广泛的，从人类到自然，从社会到私人，它试图涵盖世界的全貌。
	3.	美的民主性：惠特曼强调每个人和每件事都可以是美的，摄影通过捕捉事物的本质，将这种民主性具体化。

然而，桑塔格认为这种摄影理念是绝望的，主要基于以下原因：
	1.	理想主义与现实的冲突：
惠特曼式的包容和平等更多是一种诗意的理想，但现实世界充满了不平等和矛盾。摄影虽然试图以平等的方式记录一切，但这实际上并不能改变事物本身的地位和意义。摄影只能展示，而不能解决社会问题。
	2.	对意义的消解：
当摄影试图记录所有事物时，它会模糊重要性与不重要性之间的界限。在这一过程中，个体与社会的冲突、痛苦和不公可能被平等化为无差别的视觉信息。这种对一切事物的无差别关注可能会让人对意义感到麻木和绝望。
	3.	孤立的观看：
惠特曼式摄影强调包容性，但在现代社会的视觉文化中，摄影的观看常常是碎片化和孤立的。观者面对摄影作品时，往往缺乏更广泛的历史和社会语境，这使得摄影中的“平等性”成为一种无力的空洞姿态。
	4.	无所不在的冷漠：
摄影虽然记录了世界的复杂性，但观众在面对这些影像时可能会产生一种冷漠和距离感，因为摄影将生活转化为一系列视觉瞬间，而这些瞬间的意义往往无法深入触动人心。

因此，桑塔格认为，尽管惠特曼式的摄影理念试图以平等和包容的方式呈现世界，但它最终可能陷入一种徒劳和绝望的境地——因为它既不能完全实现惠特曼的理想，也无法解决影像本身的局限性和观看者的疏离感。这种绝望是摄影作为媒介在理想与现实之间的矛盾中难以克服的核心问题。

在《论摄影》中，苏珊·桑塔格对超现实主义的理解与摄影的媒介特性密切相关。她认为，摄影本身具有一种内在的超现实性，因为它能够以独特的方式捕捉现实，同时将现实转化为一种超越日常经验的视觉符号。以下是书中对超现实的几个关键理解：

1. 摄影作为现实与幻想的结合

桑塔格认为，摄影能够使现实本身看起来陌生化。它通过取景、构图、光影等手段，将普通的场景赋予某种戏剧性或异质性，使现实看起来不再是我们熟悉的那种样子。摄影因此具备了一种“视觉的奇迹感”，这与超现实主义在艺术中追求将现实推向梦幻、荒诞的目标相契合。

2. 摄影的证据性与神秘感的二重性

摄影的核心特质在于它既是一种对现实的记录，又能够制造不确定性和神秘感。它捕捉到的是真实的物质世界，却能通过选取某一瞬间、某个角度，将这份真实转化为一种看似不真实的感觉。这种二重性使摄影天然地接近超现实主义所追求的“真实中的不真实”。

3. 时间与永恒的对立

桑塔格指出，摄影通过将一个瞬间定格，可以冻结时间，创造出一种看似超越时间的永恒感。而这种永恒感恰恰与生活中时间的流动性形成对照，带来一种超现实的体验。这种“凝固的现实”突出了摄影在呈现真实的同时，又剥夺了真实的动态性的矛盾性。

4. 对偶然性的迷恋

超现实主义艺术强调对偶然性和不协调事物的追求，而摄影正是偶然性的媒介。摄影师无法完全预料镜头下的场景发展，这使得摄影常常能够捕捉到意料之外的细节或奇异瞬间，从而创造出超越现实逻辑的画面。这种偶然性使摄影与超现实主义的美学观念高度吻合。

5. 摄影与梦境的关联

桑塔格提到，摄影能够将现实分解为独立的图像，而这些图像在脱离了原本的语境后，可能会变得像梦境一样不可解或超乎逻辑。超现实主义强调将梦与现实结合，而摄影则以其天然的“片段性”完美地承担了这一功能。

6. 摄影作为超现实主义的工具

摄影不仅具备超现实主义的特质，而且也成为超现实主义运动的重要媒介之一。在20世纪，摄影被艺术家用于挑战传统的视觉认知，颠覆既有的现实观念。桑塔格提到，超现实主义的摄影师通过拼接、双重曝光等技术手段，进一步凸显摄影的超现实特质。

总结

在桑塔格看来，摄影并非单纯地记录现实，而是一种能够改变现实感知的媒介。它既通过对现实的捕捉而保持客观性，又通过视角、技巧以及时间的定格，赋予现实一种超现实的意味。超现实主义由此成为摄影的一种内在属性，同时也是摄影作品得以超越平庸、进入艺术领域的关键。
